Functions are the central building blocks of mathematical modelling. Whenever one quantity depends on another—such as the volume of a balloon depending on time, or the cost of producing items depending on the number manufactured—we use functions to describe this relationship formally.

---

## 1. The Concept of a Function

At its core, a function is a mathematical rule that maps an input value to exactly one output value. You can think of it as a machine: you feed it a number, the machine processes it according to a specific rule, and it spits out a new number.

<Aside type="note" title="The Hierarchy of Function Notation">
To work with functions effectively, you must be comfortable with standard function notation:
1. **The Name:** Usually a letter like $f$, $g$, or $h$. In real-world modelling, we often use meaningful letters, like $C$ for Cost or $v$ for velocity.
2. **The Input:** The variable inside the brackets, typically $x$, $t$ (time), or $n$ (number of items).
3. **The Output:** The entire expression, such as $f(x)$, represents the final value produced by the function.
</Aside>

For example, $C(n)$ might represent the cost, $C$, of producing $n$ items. If $C(n) = 50n + 200$, then evaluating $C(10)$ means calculating the cost of producing $10$ items.

## 2. Domain and Range

Not every number can be put into every function, and a function might not be capable of producing every possible number as an output. 

<Aside type="tip" title="Definitions: Domain and Range">
* **Domain:** The set of all possible valid input values (usually $x$-values) for which the function is defined.
* **Range:** The set of all possible output values (usually $y$-values or $f(x)$) that the function can produce.
</Aside>

A graph is incredibly helpful in visualising the domain and range. By looking at how far the graph stretches horizontally (left to right), you find the domain. By looking at how far it stretches vertically (bottom to top), you find the range.

Consider the function:

$$
f(x) = \sqrt{2 - x}
$$

You cannot take the square root of a negative number in the real number system. Therefore, the expression inside the square root must be greater than or equal to zero ($2 - x \ge 0$), which means $x \le 2$. 
* The **domain** is $x \le 2$.
* Because a principal square root always produces a non-negative result, the **range** is $f(x) \ge 0$.

## 3. Inverse Functions

If a function $f$ transforms an input $x$ into an output $y$, an **inverse function** reverses or "undoes" this effect, taking $y$ back to $x$. We use the notation $f^{-1}(x)$ to represent the inverse of $f(x)$.

<Aside type="tip" title="Theorems of Inverse Functions">
1. **One-to-One Requirement:** An inverse function only exists if the original function is "one-to-one" (meaning every unique output comes from exactly one unique input).
2. **Swapping Domain and Range:** The domain of the inverse function $f^{-1}(x)$ is exactly equal to the range of the original function $f(x)$, and vice versa.
</Aside>

<SteveTip title="Solving Equations using Inverses">
Understanding the relationship between a function and its inverse can save you time. Solving the equation $f(x) = 10$ is mathematically equivalent to finding $f^{-1}(10)$. You are simply asking: "What input gives me an output of $10$?"
</SteveTip>

### Graphical Representation of an Inverse

Geometrically, the graph of an inverse function $f^{-1}(x)$ is a **reflection** of the original function $f(x)$ in the diagonal line $y = x$. If a point $(a, b)$ lies on the graph of $f(x)$, then the swapped point $(b, a)$ will lie on the graph of $f^{-1}(x)$.

<Aside type="caution" title="Common Misconception">
Do not confuse the inverse function notation $f^{-1}(x)$ with the reciprocal of a number. 
$f^{-1}(x)$ **does not** mean $\frac{1}{f(x)}$. It is a specific notation used exclusively to represent the "undoing" of the function $f$.
</Aside>

    A plumber charges a fixed call-out fee of $50\text{ USD}$, plus $30\text{ USD}$ for every hour of work. The total cost, $C$, of hiring the plumber for $t$ hours is modelled by the function $C(t) = 30t + 50$.
    
    a) Calculate $C(3)$ and interpret its meaning.  
    b) Find the value of $t$ such that $C(t) = 215$.  
    c) In the context of this real-world model, write down a sensible domain and range for the function.
  </TabItem>
  <TabItem label="✅ Solution 1">
    **a)** Substitute $t = 3$ into the function:
    
    $$
    C(3) = 30(3) + 50 = 90 + 50 = 140
    $$
    
    Interpretation: The total cost of hiring the plumber for $3$ hours of work is $140\text{ USD}$.

    **b)** Set the function equal to $215$ and solve for $t$:
    
    $$
    30t + 50 = 215
    $$
    
    $$
    30t = 165
    $$
    
    $$
    t = 5.5
    $$
    
    The plumber worked for $5.5$ hours.

    **c)** Since time worked cannot be negative, $t$ must be greater than or equal to $0$. 
    **Domain:** $t \ge 0$.
    If $t \ge 0$, the minimum cost occurs when $t = 0$ (just the call-out fee). 
    **Range:** $C(t) \ge 50$.
  </TabItem>

  <TabItem label="📝 Question 4: Finding an Inverse Function Algebraically">
    Let $f(x) = \frac{2x + 5}{3}$.
    Find an expression for $f^{-1}(x)$.
  </TabItem>
  <TabItem label="✅ Solution 4">
    <Steps>
    1. Rewrite the function with $y$ instead of $f(x)$:
       $$ y = \frac{2x + 5}{3} $$
    2. Swap the $x$ and $y$ variables (this reflects the function across $y = x$):
       $$ x = \frac{2y + 5}{3} $$
    3. Rearrange the equation to make $y$ the subject:
       $$ 3x = 2y + 5 $$
       $$ 3x - 5 = 2y $$
       $$ y = \frac{3x - 5}{2} $$
    4. Replace $y$ with the inverse function notation:
       $$ f^{-1}(x) = \frac{3x - 5}{2} $$
    </Steps>
  </TabItem>
</Tabs>

    A one-to-one function $h(x)$ passes through the point $(7, -2)$.
    
    a) Write down the value of $h^{-1}(-2)$.  
    b) If the domain of $h(x)$ is $0 \le x \le 10$, write down the range of $h^{-1}(x)$.

    **b)** The range of an inverse function $h^{-1}(x)$ is exactly equal to the domain of the original function $h(x)$. 
    Therefore, the range of $h^{-1}(x)$ is $0 \le y \le 10$ (or $0 \le h^{-1}(x) \le 10$).
