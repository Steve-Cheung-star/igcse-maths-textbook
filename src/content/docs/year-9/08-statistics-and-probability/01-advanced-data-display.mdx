## 1. Inquiry: The Shape of Data
**Inquiry Question:** *Why do we need different types of graphs for the same set of numbers? How does the "shape" of a graph tell us about a population?*

In Year 9, we move beyond simple bar charts. We use specific tools to see every individual data point or to see how data is "spread out" across the whole group.

---

## 2. Stem-and-Leaf Plots
A **Stem-and-Leaf Plot** is a unique way to show the actual raw data while still grouping it into categories (like tens and units).



* **Stem:** Usually the tens digit.
* **Leaf:** The units digit (must be in ascending order).
* **Key:** Always include a key (e.g., $2 | 5 = 25$).
* **Back-to-Back:** We use these to compare two groups (e.g., Boys vs. Girls test scores).

---

## 3. Box Plots (Box-and-Whisker)
A **Box Plot** summarizes a data set using 5 key numbers (the "Five-Number Summary"):
1.  **Minimum:** The lowest value.
2.  **Lower Quartile (Q1):** The 25% mark.
3.  **Median (Q2):** The middle value (50% mark).
4.  **Upper Quartile (Q3):** The 75% mark.
5.  **Maximum:** The highest value.

### Q2: Interpreting a Box Plot
<Tabs>
  <TabItem label="📝 Question">
    If the "box" in a box plot is very wide, what does that tell you about the middle 50% of the data?
  </TabItem>
  <TabItem label="✅ Solution">
    It means there is a **large spread** in the middle half of the data (the values are very different from each other).
  </TabItem>
