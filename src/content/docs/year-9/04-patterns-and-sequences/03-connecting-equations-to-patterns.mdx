## 1. Inquiry: Patterns in Words vs. Symbols
**Inquiry Question:** *How can we translate a verbal pattern rule into a mathematical one, and how do we use it to find specific "positions" in a sequence?* 

The curriculum requires you to move fluently between three ways of seeing a pattern:
1.  **Word Rules:** "Start at 1 and multiply by 2." 
2.  **The Sequence:** $1, 2, 4, 8, 16, \dots$ 
3.  **The Algebraic Equation:** $t = 2^{n-1}$ (or for linear, $t = 2n - 1$). 

---

## 2. Objective: Predicting Terms from Equations
A key objective is determining a term given its **term number** ($n$) using an algebraic equation. 

**Example (Sample Problem from Outline):**
Given the equation $t = 2n - 1$, find the $100^{th}$ term. 
* **Step:** Substitute $n = 100$.
* **Calculation:** $t = 2(100) - 1 = 200 - 1 = \mathbf{199}$. 

---

## 3. Objective: Finding the "Position" (Solving for $n$)
You must also make connections between **solving equations** and determining the **term number** in a pattern. 

**Example:**
For a pattern with the general term $2n + 1$, which term is equal to $17$? 
* **Set up the equation:** $2n + 1 = 17$. 
* **Solve:**
    * $2n = 16$
    * $n = 8$
* **Conclusion:** $17$ is the **$8^{th}$ term** in the sequence. 

---

## 4. Objective: Comparing Pattern Rules
The outline requires you to compare rules that grow by **adding/subtracting a constant** versus those that grow by **multiplying/dividing by a constant**. 

| Growth Type | Action | Sequence Example | Name |
| :--- | :--- | :--- | :--- |
| **Constant Addition** | $+3$ | $2, 5, 8, 11 \dots$ | Arithmetic Sequence  |
| **Constant Multiplier** | $\times 2$ | $1, 2, 4, 8 \dots$ | Geometric Sequence  |

<Tabs>

    In the pattern $t = 5n - 2$, which term number is $48$? 
  </TabItem>
  <TabItem label="✅ Solution">
