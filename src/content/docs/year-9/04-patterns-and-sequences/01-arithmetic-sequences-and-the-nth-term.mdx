## 1. Inquiry: The Architecture of Patterns
**Inquiry Question:** *How can we use mathematical expressions to represent and generalize patterns?*

An **Arithmetic Sequence** is a pattern of numbers where the difference between consecutive terms is constant. We call this the **common difference ($d$)**. If you can find the rule, you can predict the $1,000^{th}$ term without writing out the whole list.

---

## 2. Finding the $n^{th}$ Term Rule
The $n^{th}$ term is a formula where $n$ represents the "position" of the number (1st, 2nd, 3rd...).

**The Formula Method:**
For a linear sequence: $\text{Rule} = dn + c$
* **$d$**: The common difference (what it goes up or down by).
* **$c$**: The "Zero Term" (what the number would be at position 0).

**Example: $5, 8, 11, 14, \dots$**
1.  **Find $d$:** $8 - 5 = 3$. So the rule starts with $3n$.
2.  **Find $c$:** Work backward from the 1st term ($5 - 3 = 2$).
3.  **Result:** The $n^{th}$ term is $3n + 2$.

*Check:* For the 4th term ($n=4$): $3(4) + 2 = 14$. It works!



---

## 3. Using the Rule
Once you have the $n^{th}$ term, you can solve two types of problems:
1.  **Find a specific term:** "What is the $50^{th}$ term?" $\rightarrow$ Substitute $n=50$.
2.  **Check if a number is in the sequence:** "Is 100 in the sequence?" $\rightarrow$ Set the rule equal to 100 and solve for $n$. If $n$ is a whole number, it's in the sequence.

---

## 4. Geometric Sequences (Intro)
In Year 9, we also look at **Geometric Sequences**, where you multiply by a constant ratio instead of adding.
* *Example:* $2, 4, 8, 16 \dots$ (Multiplying by 2).
* These grow much faster and are related to the exponents we studied in Topic 1.

### Q1: Finding the Rule
<Tabs>
  <TabItem label="📝 Question">
    Find the $n^{th}$ term for the sequence: $7, 12, 17, 22, \dots$
  </TabItem>
  <TabItem label="✅ Solution">
    1.  Difference ($d$) is $+5$. Rule starts $5n$.
    2.  Zero term ($c$): $7 - 5 = 2$.
    **Result:** $5n + 2$
  </TabItem>
</Tabs>

### Q2: Solving for n
<Tabs>
  <TabItem label="📝 Question">
    Using the rule $4n - 3$, find the $20^{th}$ term.
  </TabItem>
  <TabItem label="✅ Solution">
    Substitute $n = 20$:
    $4(20) - 3 = 80 - 3 = \mathbf{77}$
  </TabItem>
</Tabs>

### Q3: Membership Check
<Tabs>
  <TabItem label="📝 Question">
    Is the number $50$ a term in the sequence $3n + 1$?
  </TabItem>
  <TabItem label="✅ Solution">
    $3n + 1 = 50$
    $3n = 49$
    $n = 16.33...$
    Since $n$ is not a whole number, **50 is not in the sequence**.
  </TabItem>
</Tabs>
