title: 5.2 Increasing and Decreasing Functions
description: Understand how the sign of the first derivative determines whether a function is increasing, decreasing, or stationary, and learn to identify these intervals graphically and algebraically.
---

Calculus allows us to analyse the behaviour of dynamic systems. By examining the gradient of a curve at any given point, we can determine whether the overall function is rising, falling, or completely flat. This concept is fundamental to optimising outcomes in business, predicting physical motion, and understanding natural phenomena.

## 1. Graphical Interpretation of the First Derivative

The first derivative, $f'(x)$, represents the gradient (or slope) of the tangent to the curve $y = f(x)$ at any given point. 

By looking at the sign (positive, negative, or zero) of $f'(x)$, we can immediately tell what the original function $f(x)$ is doing:

* **Increasing Function ($f'(x) > 0$):** The curve is going uphill from left to right. A positive gradient means that as $x$ increases, $y$ also increases.
* **Decreasing Function ($f'(x) < 0$):** The curve is going downhill from left to right. A negative gradient means that as $x$ increases, $y$ decreases.
* **Stationary Point ($f'(x) = 0$):** The curve is momentarily flat. The tangent line is perfectly horizontal. This typically happens at the "peaks" (local maximums) or "valleys" (local minimums) of a curve.

<Aside type="caution" title="Common Pitfall: The Value vs. The Gradient">
  Do not confuse the value of the function $f(x)$ with its derivative $f'(x)$. A curve can be below the $x$-axis (meaning $f(x)$ is negative) but travelling upwards (meaning $f'(x)$ is positive). Always ask yourself: "Am I looking at the height of the curve, or the steepness of the curve?"
</Aside>

### Worked Example: Graphical Interpretation
**Question:** The graph of $y = f(x)$ has a local maximum at $x = -2$ and a local minimum at $x = 4$. State the intervals for which $f'(x) < 0$.

**Solution:**
<Steps>
  1. Visualise or sketch the curve. It hits a peak at $x = -2$ and a valley at $x = 4$.
  2. To get from a peak down to a valley, the curve must be travelling downwards. 
  3. A downwards curve means the function is decreasing, which corresponds to $f'(x) < 0$.
  4. This occurs between the two $x$-values.
     $$ \text{Interval: } -2 < x < 4 $$
</Steps>

## 2. Identifying Intervals Algebraically

When you are given the derivative function $f'(x)$ algebraically, you can find the intervals of increase and decrease using a **sign diagram** (also known as a number line test).

To build a sign diagram:
1. Find where the gradient is flat by setting $f'(x) = 0$. These $x$-values are called **critical values**.
2. Mark these critical values on a number line. This splits the domain into distinct intervals.
3. Pick a test value in each interval and substitute it into $f'(x)$.
4. If the result is positive, the function is increasing on that interval. If negative, it is decreasing.

### Worked Example: Using a Sign Diagram
**Question:** The derivative of a function is given by $f'(x) = x^2 - 4x - 5$. Find the intervals where the original function $f(x)$ is increasing.

**Solution:**
<Steps>
  1. Set the derivative equal to zero to find the critical values.
     $$
     \begin{aligned}
     x^2 - 4x - 5 &= 0 \\
     (x - 5)(x + 1) &= 0
     \end{aligned}
     $$
     The critical values are $x = -1$ and $x = 5$.
  2. Set up a number line with $-1$ and $5$, creating three intervals: $x < -1$, $-1 < x < 5$, and $x > 5$.
  3. Test a value in each interval by substituting it into $f'(x)$:
     * For $x = -2$: $f'(-2) = (-2)^2 - 4(-2) - 5 = 4 + 8 - 5 = 7$ (Positive)
     * For $x = 0$: $f'(0) = (0)^2 - 4(0) - 5 = -5$ (Negative)
     * For $x = 6$: $f'(6) = (6)^2 - 4(6) - 5 = 36 - 24 - 5 = 7$ (Positive)
  4. The function $f(x)$ is increasing where $f'(x) > 0$. Based on our test, this happens in the first and third intervals.
     $$ \text{Increasing for: } x < -1 \text{ and } x > 5 $$
</Steps>

<SteveTip title="GDC Skills: Finding Intervals Fast">
  If you are in a Paper 2 exam and are asked where a function is increasing/decreasing:
  1. Graph the **derivative** function $y = f'(x)$ directly on your TI-Nspire.
  2. Use `Menu -> Analyze Graph -> Zero` to find where it crosses the $x$-axis.
  3. Look at where the graph is *above* the $x$-axis (these are the increasing intervals of the original function) and *below* the $x$-axis (these are the decreasing intervals).
  This entirely skips the need for a manual sign diagram!
</SteveTip>

<Aside type="danger" title="Exam Trap: The Derivative Graph">
  IB examiners love giving you the graph of $y = f'(x)$ (the derivative) and asking questions about the original function $f(x)$. 
  If you are looking at an $f'(x)$ graph, **do not** look at its peaks and valleys to find where $f(x)$ is increasing/decreasing. Instead, look at where the $f'(x)$ graph is positive (above the $x$-axis) and negative (below the $x$-axis).
</Aside>

<Tabs>
  <TabItem label="📝 Q1: Reading a Curve">
    The graph of a function $f(x)$ is defined for $-5 \leq x \leq 5$. The curve starts at $(-5, -2)$, rises to a peak at $(-2, 4)$, falls to a valley at $(3, -1)$, and then rises to $(5, 6)$.  
    **a)** State the interval(s) where $f'(x) > 0$.  
    **b)** State the $x$-values where $f'(x) = 0$.  
    **c)** Is $f'(0)$ positive or negative? Give a reason for your answer.
  </TabItem>
  <TabItem label="✅ Solution 1">
    **a)**
    $f'(x) > 0$ means the function is increasing (going upwards). This happens from the start until the peak, and from the valley to the end.
    $$ -5 < x < -2 \text{ and } 3 < x < 5 $$

    **b)**
    $f'(x) = 0$ at the stationary points (the peak and the valley).
    $$ x = -2 \text{ and } x = 3 $$

    **c)**
    At $x = 0$, the curve is between the peak at $x = -2$ and the valley at $x = 3$. During this interval, the curve is travelling downwards.
    Therefore, $f'(0)$ is **negative** because the function is decreasing at that point.
  </TabItem>
</Tabs>

  <TabItem label="📝 Q2: The Derivative Graph Trap">
    The graph below shows the **derivative** function, $y = f'(x)$, for a function $f$. The graph of $f'(x)$ crosses the $x$-axis at $x = 1$ and $x = 5$. It has a local minimum at $(3, -4)$.  
    **a)** Find the interval(s) where the original function $f(x)$ is decreasing.  
    **b)** Explain why the original function $f(x)$ has a stationary point at $x = 1$.
  </TabItem>
  <TabItem label="✅ Solution 2">
    **a)**
<Steps>
  1. The original function $f(x)$ is decreasing when its derivative is negative, $f'(x) < 0$.
  2. On the graph of the derivative $y = f'(x)$, this corresponds to the regions where the graph is **below** the $x$-axis.
  3. The graph of $f'(x)$ is below the $x$-axis between its $x$-intercepts.
     $$ 1 < x < 5 $$
</Steps>

    **b)**
    A stationary point occurs when the gradient is zero. Since the graph shown is $f'(x)$, the $x$-intercepts represent where $f'(x) = 0$. The graph crosses the $x$-axis at $x = 1$, so $f'(1) = 0$, making it a stationary point.
  </TabItem>

  <TabItem label="📝 Q3: Algebraic Intervals">
    The derivative of a continuous function $g(x)$ is given by $g'(x) = -3x^2 + 12x$.  
    **a)** Find the critical values of $g(x)$.  
    **b)** Hence, determine the interval(s) where $g(x)$ is increasing.
  </TabItem>
  <TabItem label="✅ Solution 3">
    **a)**
<Steps>
  1. Set the derivative equal to zero.
     $$
     \begin{aligned}
     -3x^2 + 12x &= 0 \\
     -3x(x - 4) &= 0
     \end{aligned}
     $$
  2. Solve for $x$.
     $$ x = 0 \text{ and } x = 4 $$
</Steps>

    **b)**
<Steps>
  1. Create a sign diagram with the intervals $x < 0$, $0 < x < 4$, and $x > 4$.
  2. Test a value in each interval using $g'(x) = -3x(x - 4)$:
     * $x = -1$: $g'(-1) = -3(-1)(-1 - 4) = 3(-5) = -15$ (Negative)
     * $x = 1$: $g'(1) = -3(1)(1 - 4) = -3(-3) = 9$ (Positive)
     * $x = 5$: $g'(5) = -3(5)(5 - 4) = -15(1) = -15$ (Negative)
  3. The function is increasing where $g'(x) > 0$.
     $$ 0 < x < 4 $$
</Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Q4: Economics Context">
    A manufacturer's revenue, $R$, in USD, from selling $x$ luxury watches is modelled by a function where the marginal revenue is given by $R'(x) = 1500 - 5x$.  
    **a)** Find the number of watches that need to be sold for the marginal revenue to be zero.  
    **b)** Determine the range of sales $x$ ($x \geq 0$) for which the total revenue is increasing.
  </TabItem>
  <TabItem label="✅ Solution 4">
    **a)**
<Steps>
  1. Set the marginal revenue equal to zero.
     $$
     \begin{aligned}
     1500 - 5x &= 0 \\
     5x &= 1500 \\
     x &= 300
     \end{aligned}
     $$
  2. State the final answer clearly in context.
     $$ 300 \text{ watches need to be sold.} $$
</Steps>

    **b)**
<Steps>
  1. Revenue is increasing when the marginal revenue is positive ($R'(x) > 0$).
     $$
     \begin{aligned}
     1500 - 5x &> 0 \\
     1500 &> 5x \\
     300 &> x
     \end{aligned}
     $$
  2. The question restricts $x \geq 0$. Combine the inequalities.
     $$ 0 \leq x < 300 $$
</Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Q5: True or False">
    Consider a function $h(x)$. Determine if the following statements are True or False. If False, briefly explain why.  
    **a)** If $h(3) = -5$, then the function must be decreasing at $x = 3$.  
    **b)** If $h'(x) < 0$ for all real numbers, the graph of $h(x)$ will never cross the $y$-axis.  
    **c)** If $h'(4) = 0$ and $h'(5) = 2$, the function is increasing at $x = 5$.
  </TabItem>
  <TabItem label="✅ Solution 5">
    **a)** **False.** The value $h(3) = -5$ only tells us the height of the function (it is below the $x$-axis). It tells us nothing about its gradient $h'(3)$, so we do not know if it is increasing or decreasing.  
    
    **b)** **False.** If $h'(x) < 0$, the function is always decreasing (sloping downwards). A line that slopes downwards from left to right across the entire real number line will definitively cross the $y$-axis at some point.  
    
    **c)** **True.** The value of the derivative at $x = 5$ is positive ($2 > 0$), which is the exact definition of the function increasing at that point.
  </TabItem>
</Tabs>

    * At $x = -2$, $f'(x) = 0$.

  2. Analyse the behaviour around $x = 3$: The function changes from decreasing (negative derivative) to increasing (positive derivative). This means the curve goes down, flattens out, and then goes up. This forms a **local minimum (valley)** at $x = 3$.

  <TabItem label="📝 Q7: Kinematics (Physics Context)">
    A drone is moving vertically. Its height, $h$ metres, above the ground at time $t$ seconds ($t \geq 0$) is monitored. The rate of change of its height (its velocity) is given by $\dfrac{dh}{dt} = 8 - 2t$.  
    **a)** At what time does the drone stop momentarily?  
    **b)** Find the time interval during which the drone's height is decreasing. What does this mean in real life?
  </TabItem>
  <TabItem label="✅ Solution 7">
    **a)**
<Steps>
  1. The drone stops momentarily when its velocity (rate of change of height) is zero.
     $$
     \begin{aligned}
     8 - 2t &= 0 \\
     2t &= 8 \\
     t &= 4 \text{ seconds}
     \end{aligned}
     $$
</Steps>

    **b)**
<Steps>
  1. The height is decreasing when the velocity is negative ($\dfrac{dh}{dt} < 0$).
     $$
     \begin{aligned}
     8 - 2t &< 0 \\
     8 &< 2t \\
     4 &< t
     \end{aligned}
     $$
  2. This means the height is decreasing for $t > 4$. In real life context, this means the drone has finished its ascent and is now **descending (falling) towards the ground**.
</Steps>
  </TabItem>

  <TabItem label="📝 Q8: Table of Values">
    A table of values is provided for a continuous, smooth function's derivative, $f'(x)$.  

    | $x$ | $-2$ | $-1$ | $0$ | $1$ | $2$ | $3$ |
    | :--- | :---: | :---: | :---: | :---: | :---: | :---: |
    | $f'(x)$ | $-4$ | $-1$ | $0$ | $2$ | $5$ | $9$ |

    **a)** State the interval (from the given $x$-values) over which the function $f(x)$ is increasing.  
    **b)** A student claims that because $f'(x)$ is getting larger across the whole table ($-4$ to $9$), the original function $f(x)$ must be increasing across the whole table. Explain why the student is incorrect.
  </TabItem>
  <TabItem label="✅ Solution 8">
    **a)**
    The function $f(x)$ is increasing when its derivative $f'(x) > 0$. Looking at the table, $f'(x)$ is positive for $x = 1, 2, 3$.
    Therefore, based on the provided table, the function is increasing for $x > 0$.

    **b)**
    The student is confusing the *value* of the derivative with the *rate of change* of the derivative. 
    While the values in the $f'(x)$ row are mathematically increasing (getting larger), some of those values are still negative (e.g., at $x = -2$, $f'(x) = -4$). 
    A negative derivative, even if it is becoming "less negative", still means the original function $f(x)$ is decreasing. The function $f(x)$ only starts increasing once $f'(x)$ actually crosses zero and becomes positive.
  </TabItem>
