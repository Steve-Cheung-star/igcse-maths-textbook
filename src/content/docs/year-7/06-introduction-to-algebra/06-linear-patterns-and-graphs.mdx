## 1. Inquiry: Seeing the Growth
**Inquiry Questions:** *How can we predict the 100th term of a pattern without drawing it? How does a constant change look on a graph?* 

In algebra, a **growing pattern** is a sequence that increases by the same amount each time. We can use a table of values to show the relationship between the **term number** ($n$) and the **term value**.



---

## 2. Developing the General Term
To find the "rule" for a pattern, we look for the **common difference** (the amount added each time). This becomes the coefficient of our variable $n$.

**Example:**
Pattern: 3, 5, 7, 9...
1. **Find the difference:** Each step adds 2. So the rule starts with $2n$.
2. **Adjust the rule:** If $n=1$, $2(1) = 2$. To get to our first term (3), we need to add 1.
3. **General Term:** $2n + 1$.

---

## 3. Graphing the Pattern
Once you have a table of values, you can treat each pair as a coordinate $(x, y)$ and plot it on a graph. A linear pattern will always form a **straight line**.

| Term Number ($x$) | Term Value ($y$) | Coordinate |
| :--- | :--- | :--- |
| 1 | 3 | (1, 3) |
| 2 | 5 | (2, 5) |
| 3 | 7 | (3, 7) |

## 4. Practice & Homework

### Q1: Extending a Pattern
<Tabs>
  <TabItem label="📝 Question">
    A pattern follows the rule $4n - 1$. Calculate the value for:
    
    a) The 1st term
    
    b) The 5th term
    
    c) The 10th term
  </TabItem>
  <TabItem label="✅ Solution">
    a) $4(1) - 1 = \mathbf{3}$ 
    
    b) $4(5) - 1 = \mathbf{19}$ 
    
    c) $4(10) - 1 = \mathbf{39}$ 
  </TabItem>
</Tabs>

### Q2: Finding the Rule
<Tabs>
  <TabItem label="📝 Question">
    Find the general term (rule) for this sequence: 5, 8, 11, 14...
  </TabItem>
  <TabItem label="✅ Solution">
    1. Difference is +3, so we start with $3n$.
    2. Check $n=1$: $3(1) = 3$. We need 5, so add 2.
    3. **Rule:** $3n + 2$.
  </TabItem>
</Tabs>

### Q3: Graphing Coordinates
<Tabs>
  <TabItem label="📝 Question">
    If a pattern follows $y = x + 2$, what are the coordinates for the first three terms?
  </TabItem>
  <TabItem label="✅ Solution">
    Term 1: $1+2=3 \rightarrow \mathbf{(1, 3)}$
    
    Term 2: $2+2=4 \rightarrow \mathbf{(2, 4)}$
    
    Term 3: $3+2=5 \rightarrow \mathbf{(3, 5)}$ 
  </TabItem>
</Tabs>
