## 1. Inquiry: The Balanced Scale
**Inquiry Question:** *What is the best way to solve linear equations?* 

Think of an equation as a balanced scale. The equals sign ($=$) tells us that the value on the left is exactly the same as the value on the right. To keep the scale balanced, **whatever you do to one side, you must do to the other.**



---

## 2. One-Step Equations
To solve an equation, we must "undo" the operation attached to the variable using its **inverse (opposite) operation**.

| Original Operation | Inverse Operation |
| :--- | :--- |
| Addition ($+$) | Subtraction ($-$) |
| Subtraction ($-$) | Addition ($+$) |
| Multiplication ($\times$) | Division ($\div$) |
| Division ($\div$) | Multiplication ($\times$) |

**Example 1 (Addition):**
$x + 7 = 15$
* Subtract 7 from both sides: $x = 15 - 7$
* **Result:** $x = 8$

**Example 2 (Multiplication):**
$3x = 12$
* Divide both sides by 3: $x = 12 \div 3$
* **Result:** $x = 4$

---

## 3. Two-Step Equations
Some equations require two steps to solve. Always "undo" the addition or subtraction first, then the multiplication or division.

**Example:**
$2x + 5 = 13$
1. **Subtract 5** from both sides: $2x = 8$
2. **Divide by 2** on both sides: $x = 4$
