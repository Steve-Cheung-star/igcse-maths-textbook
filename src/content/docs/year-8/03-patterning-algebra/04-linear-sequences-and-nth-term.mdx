## 1. Inquiry: The Pattern Code
**Inquiry Questions:** *If a pattern starts 5, 8, 11..., how can we calculate the 100th number without adding 3 over and over again? What is the relationship between a sequence and a linear equation?*

In Year 7, you learned to find the "term-to-term" rule (e.g., +3). In Year 8, we find the **position-to-term rule**, known as the $n^{th}$ term. This formula allows us to calculate any number in a sequence based on its position ($n$).

---

## 2. Finding the $n^{th}$ Term Formula
For a linear sequence (where the difference is always the same), we use the formula structure:
$$n^{th} \text{ term} = dn + c$$

* **$d$ (The Common Difference):** How much the sequence goes up or down by each time.
* **$c$ (The "Zero" Term):** The value that would come *before* the first term in the sequence.

**Example: 5, 7, 9, 11...**
1.  **Find $d$:** The sequence goes up by 2 each time. So, $d = 2$. This gives us $2n$.
2.  **Find $c$:** Work backward from the first term (5) by subtracting the difference (2).
    * $5 - 2 = 3$. So, $c = +3$.
3.  **The Formula:** $2n + 3$



---

## 3. Using the Formula
Once you have the code, you can predict the future.

**Example: Find the 50th term of the sequence $2n + 3$.**
1.  Replace $n$ with 50.
2.  Calculation: $2(50) + 3 = 100 + 3$.
3.  **Result:** 103.

## 4. Practice & Homework (20 Minutes)

### Q1: Identify the Rule
<Tabs>
  <TabItem label="📝 Question">
    Find the $n^{th}$ term formula for these sequences:
    
    a) 4, 7, 10, 13...
    
    b) 10, 15, 20, 25...
    
    c) 8, 6, 4, 2... (Careful: it's decreasing!)
  </TabItem>
  <TabItem label="✅ Solution">
    a) Difference is 3. Zero term is $4 - 3 = 1$. Rule: **$3n + 1$**
    
    b) Difference is 5. Zero term is $10 - 5 = 5$. Rule: **$5n + 5$**
    
    c) Difference is -2. Zero term is $8 - (-2) = 10$. Rule: **$-2n + 10$**
  </TabItem>
</Tabs>

### Q2: Predictive Math
<Tabs>
  <TabItem label="📝 Question">
    A sequence has the $n^{th}$ term $4n - 2$. 
    
    a) What is the 10th term?
    
    b) What is the 100th term?
  </TabItem>
  <TabItem label="✅ Solution">
    a) $4(10) - 2 = \mathbf{38}$
    
    b) $4(100) - 2 = \mathbf{398}$
  </TabItem>
</Tabs>

### Q3: Reverse Logic
<Tabs>
  <TabItem label="📝 Question">
    Is the number 22 a part of the sequence $3n + 1$? (Hint: Set $3n + 1 = 22$ and solve for $n$).
  </TabItem>
  <TabItem label="✅ Solution">
    1. $3n + 1 = 22$
    2. $3n = 21$
    3. $n = 7$
    **Answer:** Yes, it is the 7th term because $n$ is a whole number.
  </TabItem>
</Tabs>
