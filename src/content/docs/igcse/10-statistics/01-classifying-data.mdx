<Aside type="note" title="Types of Data">
Data generally falls into two main categories:
1. **Qualitative (Categorical) Data:** Data that is described in words (e.g., eye colour, favourite car brand, blood type).
2. **Quantitative (Numerical) Data:** Data that is measured or counted in numbers. This is further split into:
   * **Discrete:** Data that can only take specific, exact values (e.g., number of siblings, shoe size, number of cars). You count it.
   * **Continuous:** Data that can take any value within a range (e.g., height, weight, time). You measure it.
</Aside>

## 1. Tally and Frequency Tables

When collecting raw data, a **tally chart** is the most efficient way to record results as they happen. 

We use the "five-bar gate" method to keep track of numbers easily. You draw four vertical lines, and the fifth line crosses diagonally through them. This makes it incredibly easy to count up the final frequencies in multiples of five.

### Example: Ungrouped Data

A teacher asks $20$ students how many pets they own. The raw data is:
`0, 1, 1, 2, 0, 3, 1, 0, 4, 1, 2, 1, 0, 2, 1, 5, 0, 1, 2, 1`

We can organise this into a frequency table:

| Number of Pets | Tally | Frequency |
| :--- | :--- | :--- |
| $0$ | 卌 | $5$ |
| $1$ | 卌 III | $8$ |
| $2$ | IIII | $4$ |
| $3$ | I | $1$ |
| $4$ | I | $1$ |
| $5$ | I | $1$ |
| **Total** | | **20** |

*Note: Grouped tally tables can also be used for continuous data using inequalities (e.g., $0 < h \le 10$, $10 < h \le 20$). We will explore this further in later sections.*

---

## 2. Two-Way Tables

A **two-way table** displays data that belongs to **two different categories** at the same time. It is essentially a matrix where the rows represent one category and the columns represent another.

### Reading and Completing a Two-Way Table

The most important rule of a two-way table is that **the rows and columns must sum perfectly to the totals**.

**Example:** A group of $135$ students were asked whether they prefer Maths or Science. The results are split by whether the student is in Year 10 or Year 11.

| | Prefers Maths | Prefers Science | Total |
| :--- | :--- | :--- | :--- |
| **Year 10** | $41$ | $38$ | **79** |
| **Year 11** | $25$ | $31$ | **56** |
| **Total** | **66** | **69** | **135** |

<Aside type="tip" title="The Grand Total Check">
The bottom-right corner cell is the "Grand Total". It is the total number of people surveyed. The bottom row totals ($66 + 69$) MUST equal the far-right column totals ($79 + 56$). Both paths must lead to exactly $135$. If they don't, you've made an addition error!
</Aside>

  <TabItem label="📝 Question 5: Two-Way Tables and Probabilities">
    A sports club has $100$ members. They play either Tennis, Badminton, or Squash.
    * There are $45$ male members in total.
    * $12$ females play Squash.
    * $25$ members play Badminton, of which $15$ are male.
    * $38$ members play Tennis in total.
    
    a) Copy and complete the two-way table below to show this information.

    | | Tennis | Badminton | Squash | Total |
    | :--- | :--- | :--- | :--- | :--- |
    | **Male** | | | | |
    | **Female** | | | | |
    | **Total** | | | | |

    b) If a member is chosen at random, what is the probability that they are a male who plays Tennis?

    c) If a **female** member is chosen at random, what is the probability that she plays Badminton?
  </TabItem>
  <TabItem label="✅ Solution 5">
    **a)**

    <Steps>
      1. Fill in the absolute facts given in the text: Total members = $100$. Total males = $45$. Female Squash = $12$. Total Badminton = $25$. Male Badminton = $15$. Total Tennis = $38$.
      2. Calculate total females: $100 - 45 = 55$.
      3. Calculate female Badminton: $25 - 15 = 10$.
      4. Calculate total Squash: $100 - (38 + 25) = 100 - 63 = 37$.
      5. Calculate male Squash: Total Squash ($37$) - Female Squash ($12$) = $25$.
      6. Calculate male Tennis: Total males ($45$) - Male Badminton ($15$) - Male Squash ($25$) = $5$.
      7. Calculate female Tennis: Total Tennis ($38$) - Male Tennis ($5$) = $33$.
    </Steps>
    
    | | Tennis | Badminton | Squash | Total |
    | :--- | :--- | :--- | :--- | :--- |
    | **Male** | $5$ | $15$ | $25$ | **45** |
    | **Female** | $33$ | $10$ | $12$ | **55** |
    | **Total** | **38** | **25** | **37** | **100** |

    **b)**

    <Steps>
      1. Look for the specific intersection cell for Male and Tennis. The number is $5$. 
      2. The total number of members is $100$.
      3. $P(\text{Male and Tennis}) = \frac{5}{100} = \mathbf{\frac{1}{20}}$
    </Steps>
    
    **c)**

    <Steps>
      1. Because we are told a female member is chosen, the denominator is no longer the grand total ($100$); it is only the total number of females ($55$). 
      2. Out of those $55$ females, the number who play Badminton is $10$.
      3. $P(\text{Badminton given Female}) = \frac{10}{55} = \mathbf{\frac{2}{11}}$
    </Steps>

  </TabItem>
</Tabs>
