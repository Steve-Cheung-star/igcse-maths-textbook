In statistics, quantitative (numerical) data is split into two very distinct categories: **discrete** and **continuous**. Knowing which type of data you are working with is crucial, as it entirely changes how the data is grouped in tables and which types of graphs you are allowed to draw.

<Aside type="tip" title="The Golden Rule">
* If you **count** it, it is **discrete**.
* If you **measure** it, it is **continuous**.
</Aside>

---

## 1. Discrete Data

**Discrete data** can only take exact, specific values. These values are usually whole numbers, but they don't *have* to be, as long as there are clear, impossible gaps between the numbers. You cannot have half of a discrete unit in a meaningful way.

**Examples of discrete data:**
* The number of students in a classroom ($15$, $16$, $17$ — you cannot have $16.3$ students).
* The number of goals scored in a football match ($0$, $1$, $2$, $3$).
* Shoe sizes ($6$, $6.5$, $7$, $7.5$ — notice these include decimals, but you still cannot have a shoe size of $6.234$).

## 2. Continuous Data

**Continuous data** can take *any* value within a given range. It is not restricted to specific steps. The only limit to how exact continuous data can be is the accuracy of your measuring instrument. 

**Examples of continuous data:**
* **Height:** A plant might be $15\text{ cm}$ tall, or $15.2\text{ cm}$, or $15.247\text{ cm}$ if you have a precise enough laser.
* **Weight/Mass:** A bag of sugar weighing $1\text{ kg}$ might actually be $1.002\text{ kg}$.
* **Time:** The time it takes to run a race ($10.45$ seconds).
* **Temperature:** The temperature outside ($22.4^\circ\text{C}$).

<Aside type="caution" title="Is Money Discrete or Continuous?">
This is a very common trap! Because physical coins and notes have set values (e.g., $1$ pence, $5$ pence), you cannot have half a penny. Technically, money is **discrete**. However, in statistics and IGCSE questions, large ranges of money (like salaries or profits) are usually treated as **continuous data** when grouped in tables or graphs.
</Aside>

## 3. Grouping the Data in Tables

Because continuous data can be any decimal value, we have to use different notation when grouping it into a frequency table compared to discrete data.

### Grouping Discrete Data
For discrete data, we use simple ranges because there is no confusion about what happens between the numbers.

| Number of Goals | Frequency |
| :--- | :--- |
| $0 - 2$ | $5$ |
| $3 - 5$ | $12$ |
| $6 - 8$ | $4$ |

*If a team scores $2$ goals, they go in the first row. If they score $3$, they go in the second. There is no such thing as $2.5$ goals, so there is no gap in our data collection.*

### Grouping Continuous Data
For continuous data, using "$0 - 10$" and "$11 - 20$" creates a massive problem. What happens if a plant is $10.4\text{ cm}$ tall? It wouldn't fit in either group! 

Therefore, for continuous data, we **must** use inequality symbols ($\le$ and $<$).

| Height, $h$ (cm) | Frequency |
| :--- | :--- |
| $0 < h \le 10$ | $8$ |
| $10 < h \le 20$ | $15$ |
| $20 < h \le 30$ | $7$ |

*Notice how $10$ is the boundary for both rows. The inequality signs tell us exactly where a plant measuring exactly $10\text{ cm}$ goes. Because of the $\le$ sign, it goes into the **first** row. A plant that is $10.1\text{ cm}$ goes into the **second** row.*

<SteveTip title="Graphing the Data">
Later in the syllabus, you will learn to graph these. Remember this strict pairing: 
**Discrete data** is usually graphed using a **Bar Chart** (with gaps between the bars). 
**Continuous data** is graphed using a **Histogram** (where the bars touch, because the data is seamlessly continuous).
</SteveTip>

  <TabItem label="📝 Question 2: Grouping Continuous Data">
    A botanist measures the lengths of $40$ leaves to the nearest millimetre. 
    One leaf is exactly $30\text{ mm}$ long. Another is $30.5\text{ mm}$ long, and a third is $39.9\text{ mm}$ long.
    
    Which of the following classes does each leaf belong to?
    
    * **Class A:** $10 < L \le 20$  
    * **Class B:** $20 < L \le 30$  
    * **Class C:** $30 < L \le 40$

    a) Which class for the $30\text{ mm}$ leaf?

    b) Which class for the $30.5\text{ mm}$ leaf?

    c) Which class for the $39.9\text{ mm}$ leaf?
  </TabItem>
  <TabItem label="✅ Solution 2">

    <Steps>
      1. **a)** Look at the $30\text{ mm}$ leaf. The boundary $30$ appears in both Class B and Class C. Class B uses $\le 30$, which means "less than or **equal to** $30$". Therefore, the $30\text{ mm}$ leaf goes into **Class B**.
      2. **b)** Look at the $30.5\text{ mm}$ leaf. $30.5$ is strictly greater than $30$. Class C uses $30 < L$, which means "$L$ is strictly greater than $30$". Therefore, it goes into **Class C**.
      3. **c)** The $39.9\text{ mm}$ leaf is less than $40$, fitting cleanly into **Class C** ($30 < L \le 40$).
    </Steps>

  </TabItem>
</Tabs>

      2. The second row covers up to exactly $5\text{ kg}$ ($\le 5$).

    </Steps>

<Tabs>
  <TabItem label="📝 Question 7: Interpreting a Continuous Table">
    The time, $t$ minutes, taken by $50$ students to complete a math puzzle is recorded in the table below.

    | Time ($t$ minutes) | Frequency |
    | :--- | :--- |
    | $0 < t \le 5$ | $3$ |
    | $5 < t \le 10$ | $15$ |
    | $10 < t \le 15$ | $22$ |
    | $15 < t \le 20$ | $10$ |

    a) Which class interval contains the median student (the $25^{\text{th}}$ student)?

    b) How many students took strictly more than $10$ minutes?
  </TabItem>
  <TabItem label="✅ Solution 7">
    **a)**

    <Steps>
      1. To find the median student, count a running total of frequencies until you reach $25$.
      2. First class: $3$ students. Total so far = $3$.
      3. Second class: $15$ students. Total so far = $3 + 15 = 18$.
      4. Third class: $22$ students. Total so far = $18 + 22 = 40$.
      5. The $25^{\text{th}}$ student falls inside the third group. 
      6. The interval is **$10 < t \le 15$**.
    </Steps>

    **b)**

    <Steps>
      1. "Strictly more than 10 minutes" covers the last two rows ($10 < t \le 15$ and $15 < t \le 20$).
      2. Add their frequencies: $22 + 10$.
      3. The answer is **$32$ students**.
    </Steps>

  </TabItem>
</Tabs>
