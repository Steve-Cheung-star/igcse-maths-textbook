In mathematics, you can take a standard graph and "move" it around the coordinate grid. Sliding a graph up, down, left, or right without changing its fundamental shape or orientation is called a **translation**. 

In the IGCSE 0607 syllabus, you need to understand how adding or subtracting an integer $k$ to a function changes its graph, and how to describe this movement using a translation vector.

---

## 1. Vertical Translations: $y = f(x) + k$

When you add or subtract a number on the *outside* of a function, it affects the $y$-values directly. This causes the entire graph to slide vertically.

* If you have **$y = f(x) + k$** (where $k$ is positive), the graph shifts **UP** by $k$ units.
* If you have **$y = f(x) - k$** (where $k$ is positive), the graph shifts **DOWN** by $k$ units.

Let's look at the base graph of $f(x) = x^2$ and see what happens when we apply vertical translations:

## 2. Horizontal Translations: $y = f(x + k)$

When you add or subtract a number on the *inside* of the function's brackets, it affects the $x$-values before the function evaluates them. This causes the entire graph to slide horizontally.

<SteveTip title="The Opposite Direction Trap">
Horizontal translations are counter-intuitive. They do the **opposite** of what the sign suggests!
* A plus sign inside the bracket shifts the graph to the **LEFT** (the negative direction).
* A minus sign inside the bracket shifts the graph to the **RIGHT** (the positive direction).
</SteveTip>

* **$y = f(x + k)$** shifts the graph **LEFT** by $k$ units.
* **$y = f(x - k)$** shifts the graph **RIGHT** by $k$ units.

## 3. Describing Transformations using Column Vectors

In exams, you are frequently asked to describe a translation using a column vector. A column vector looks like this: $\begin{pmatrix} p \\ q \end{pmatrix}$.

* The top number ($p$) tells you the **horizontal** movement (left/right).
* The bottom number ($q$) tells you the **vertical** movement (up/down).

If a function $f(x)$ is translated by the vector $\begin{pmatrix} p \\ q \end{pmatrix}$ to become a new function $g(x)$, the formula is:
$$g(x) = f(x - p) + q$$

<Aside type="note" title="Connecting the Formula to the Vector">
Because of the "opposite direction trap" for horizontal translations, the horizontal shift ($p$) takes a minus sign inside the brackets, but the vertical shift ($q$) stays precisely as it is on the outside.

For example, translating by $\begin{pmatrix} -5 \\ 2 \end{pmatrix}$ means going left 5 and up 2. 
The new function would be $y = f(x + 5) + 2$.
</Aside>

    Consider a base function $y = f(x)$.
    
    a) Describe the geometric translation represented by $y = f(x) + 8$.

    b) Describe the geometric translation represented by $y = f(x - 3)$.

    c) Write down the new function, in terms of $f(x)$, if the graph is shifted down by $5$ units.

    d) Write down the new function, in terms of $f(x)$, if the graph is shifted left by $2$ units.
  </TabItem>
  <TabItem label="✅ Solution 1">
    **a)**
    <Steps>
      1. A number added on the outside of the function is a vertical shift.
      2. It is a translation **up by $8$ units**.
    </Steps>

    **b)**
    <Steps>
      1. A subtracted number on the inside of the function is a horizontal shift in the positive direction.
      2. It is a translation **right by $3$ units**.
    </Steps>

    **c)**
    <Steps>
      1. A downward shift requires subtracting from the outside of the function.
      2. **$y = f(x) - 5$**
    </Steps>

    **d)**
    <Steps>
      1. A leftward shift requires adding to the inside of the function brackets.
      2. **$y = f(x + 2)$**
    </Steps>
  </TabItem>

  <TabItem label="📝 Question 2: Applying Vectors">
    The base graph $f(x) = x^2$ is mapped onto a new function $g(x)$ by a specific translation vector. Find the expression for $g(x)$ if the vector is:
    
    a) $\begin{pmatrix} 6 \\ 0 \end{pmatrix}$

    b) $\begin{pmatrix} 0 \\ -4 \end{pmatrix}$

    c) $\begin{pmatrix} -2 \\ 7 \end{pmatrix}$

    d) Expand and simplify your answer from part (c) into the form $ax^2 + bx + c$.
  </TabItem>
  <TabItem label="✅ Solution 2">
    **a)**
    <Steps>
      1. $\begin{pmatrix} 6 \\ 0 \end{pmatrix}$ means right $6$ and up $0$.
      2. Inside the bracket, we subtract $6$.
      3. **$g(x) = (x - 6)^2$**
    </Steps>

    **b)**
    <Steps>
      1. $\begin{pmatrix} 0 \\ -4 \end{pmatrix}$ means right $0$ and down $4$.
      2. Outside the bracket, we subtract $4$.
      3. **$g(x) = x^2 - 4$**
    </Steps>

    **c)**
    <Steps>
      1. $\begin{pmatrix} -2 \\ 7 \end{pmatrix}$ means left $2$ and up $7$.
      2. Inside we add $2$, outside we add $7$.
      3. **$g(x) = (x + 2)^2 + 7$**
    </Steps>

    **d)**
    <Steps>
      1. Start with $g(x) = (x + 2)^2 + 7$.
      2. Expand the squared bracket: $(x + 2)(x + 2) = x^2 + 4x + 4$.
      3. Add the $7$: $x^2 + 4x + 4 + 7$.
      4. **$g(x) = x^2 + 4x + 11$**
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 3: Finding the Vector">
    For each pair of functions below, write down the column vector $\begin{pmatrix} p \\ q \end{pmatrix}$ that translates $f(x)$ onto the new function.
    
    a) $f(x) = 3^x$ translates to $g(x) = 3^x + 9$.

    b) $f(x) = \cos(x)$ translates to $h(x) = \cos(x - 45^\circ)$.

    c) $f(x) = \frac{1}{x}$ translates to $j(x) = \frac{1}{x + 5} - 2$.

    d) Briefly explain why the top number of your vector in part (c) is negative.
  </TabItem>
  <TabItem label="✅ Solution 3">
    **a)**
    <Steps>
      1. The $+9$ is outside the power, meaning it's a vertical shift up by $9$. There is no horizontal shift.
      2. Vector is **$\begin{pmatrix} 0 \\ 9 \end{pmatrix}$**.
    </Steps>

    **b)**
    <Steps>
      1. The $-45^\circ$ is inside the bracket, meaning a shift right by $45^\circ$. There is no vertical shift.
      2. Vector is **$\begin{pmatrix} 45 \\ 0 \end{pmatrix}$**.
    </Steps>

    **c)**
    <Steps>
      1. The $+5$ is inside the denominator, meaning a shift left by $5$ (so $-5$).
      2. The $-2$ is outside, meaning a shift down by $2$ (so $-2$).
      3. Vector is **$\begin{pmatrix} -5 \\ -2 \end{pmatrix}$**.
    </Steps>

    **d)**
    <Steps>
      1. The top number represents horizontal movement.
      2. **Adding $5$ inside the function shifts the graph in the negative direction (left)**, therefore the vector requires a $-5$ to represent this leftward movement.
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 4: Vertex Movement">
    The base quadratic $f(x) = x^2$ has its vertex at $(0, 0)$. It is translated to become $g(x) = (x - 8)^2 + 3$.
    
    a) State the coordinates of the vertex of $g(x)$.

    b) Write down the translation vector that maps $f(x)$ onto $g(x)$.

    c) The graph of $g(x)$ is further translated by the vector $\begin{pmatrix} -4 \\ -5 \end{pmatrix}$ to become $h(x)$. Find the coordinates of the vertex of $h(x)$.

    d) Write the algebraic equation for $h(x)$ in vertex form.
  </TabItem>
  <TabItem label="✅ Solution 4">
    **a)**
    <Steps>
      1. The equation shows a shift right by $8$ and up by $3$.
      2. The new vertex is **$(8, 3)$**.
    </Steps>

    **b)**
    <Steps>
      1. Horizontal shift is $+8$, vertical shift is $+3$.
      2. **$\begin{pmatrix} 8 \\ 3 \end{pmatrix}$**

    **c)**
    <Steps>
      1. Start with the vertex of $g(x)$ at $(8, 3)$.
      2. Apply the vector: move left $4$ (so $8 - 4 = 4$) and down $5$ (so $3 - 5 = -2$).
      3. The new vertex is **$(4, -2)$**.
    </Steps>

    **d)**
    <Steps>
      1. A quadratic with vertex $(4, -2)$ takes the form $(x - h)^2 + k$.
      2. **$h(x) = (x - 4)^2 - 2$**
    </Steps>
  </TabItem>

  <TabItem label="📝 Question 5: Asymptote Movement">
    The basic rational function $f(x) = \frac{1}{x}$ has a vertical asymptote at $x = 0$ and a horizontal asymptote at $y = 0$.
    
    a) The function is mapped onto $g(x) = \frac{1}{x + 3} + 5$. State the translation vector.

    b) State the equation of the new vertical asymptote for $g(x)$.

    c) State the equation of the new horizontal asymptote for $g(x)$.

    d) Find the exact coordinates of the y-intercept of $g(x)$.
  </TabItem>
  <TabItem label="✅ Solution 5">
    **a)**
    <Steps>
      1. $+3$ in the denominator means left $3$ ($-3$).
      2. $+5$ on the outside means up $5$ ($+5$).
      3. Vector is **$\begin{pmatrix} -3 \\ 5 \end{pmatrix}$**.
    </Steps>

    **b)**
    <Steps>
      1. A horizontal shift of $-3$ moves the vertical asymptote from $x = 0$ to $x = -3$.
      2. (Check: set denominator to zero: $x + 3 = 0 \implies x = -3$).
      3. **$x = -3$**
    </Steps>

    **c)**
    <Steps>
      1. A vertical shift of $+5$ moves the horizontal asymptote from $y = 0$ to $y = 5$.
      2. **$y = 5$**
    </Steps>

    **d)**
    <Steps>
      1. Set $x = 0$: $g(0) = \frac{1}{0 + 3} + 5$.
      2. $g(0) = \frac{1}{3} + 5 = \frac{16}{3}$ (or $5.33$).
      3. The y-intercept is **$(0, \frac{16}{3})$**.
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 6: Reversing Translations">
    A cubic function $g(x) = (x - 7)^3 - 1$ was formed by translating the base function $f(x) = x^3$.
    
    a) State the column vector that maps $f(x)$ onto $g(x)$.

    b) What column vector would translate $g(x)$ back to $f(x)$?

    c) A third function is given by $h(x) = x^3 + 4$. Write the column vector that maps $g(x)$ directly onto $h(x)$.

    d) Evaluate $h(2)$.
  </TabItem>
  <TabItem label="✅ Solution 6">
    **a)**
    <Steps>
      1. $g(x)$ is shifted right by $7$ and down by $1$.
      2. Vector is **$\begin{pmatrix} 7 \\ -1 \end{pmatrix}$**.
    </Steps>

    **b)**
    <Steps>
      1. To reverse the translation, simply flip the signs of the original vector.
      2. You must go left $7$ and up $1$.
      3. Vector is **$\begin{pmatrix} -7 \\ 1 \end{pmatrix}$**.
    </Steps>

    **c)**
    <Steps>
      1. $g(x)$ has "center" at $(7, -1)$. $h(x)$ has "center" at $(0, 4)$.
      2. To move from $x=7$ to $x=0$, move left $7$ ($-7$).
      3. To move from $y=-1$ to $y=4$, move up $5$ ($+5$).
      4. Vector is **$\begin{pmatrix} -7 \\ 5 \end{pmatrix}$**.
    </Steps>

    **d)**
    <Steps>
      1. Substitute $x = 2$ into $h(x)$:
      2. $h(2) = 2^3 + 4$
      3. $h(2) = 8 + 4 = \mathbf{12}$
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 7: Interpreting Coordinate Shifts">
    A point $P(3, 8)$ lies on the curve $y = f(x)$.
    
    a) Find the corresponding coordinates of point $P$ on the curve $y = f(x) + 6$.

    b) Find the corresponding coordinates of point $P$ on the curve $y = f(x + 4)$.

    c) The original curve is translated by the vector $\begin{pmatrix} -2 \\ -5 \end{pmatrix}$. Find the new coordinates of $P$.

    d) A different point $Q(a, b)$ on $y = f(x)$ maps to the coordinate $(10, -1)$ after a translation of $\begin{pmatrix} 3 \\ -4 \end{pmatrix}$. Find the values of $a$ and $b$.
  </TabItem>
  <TabItem label="✅ Solution 7">
    **a)**
    <Steps>
      1. $y = f(x) + 6$ is a vertical shift up by $6$.
      2. Only the $y$-coordinate changes: $8 + 6 = 14$.
      3. **$(3, 14)$**
    </Steps>

    **b)**
    <Steps>
      1. $y = f(x + 4)$ is a horizontal shift left by $4$.
      2. Only the $x$-coordinate changes: $3 - 4 = -1$.
      3. **$(-1, 8)$**
    </Steps>

    **c)**
    <Steps>
      1. Apply the vector: subtract $2$ from $x$ and subtract $5$ from $y$.
      2. $x = 3 - 2 = 1$
      3. $y = 8 - 5 = 3$
      4. **$(1, 3)$**
    </Steps>

    **d)**
    <Steps>
      1. We know $a + 3 = 10$ and $b - 4 = -1$.
      2. Solve for $a$: $10 - 3 \implies a = 7$.
      3. Solve for $b$: $-1 + 4 \implies b = 3$.
      4. **$a = 7$, $b = 3$**
    </Steps>
  </TabItem>
</Tabs>
