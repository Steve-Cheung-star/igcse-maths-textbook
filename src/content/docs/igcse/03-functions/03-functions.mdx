In mathematics, a function is like a machine: you put a number in, the machine performs a specific set of operations on it, and it spits a new number out. This chapter will teach you how to write functions, reverse them, and chain them together.

---

## 1. Functions, Domain and Range

We use **function notation** to describe these mathematical machines. Instead of writing $y = 3x - 5$, we write:
$$f(x) = 3x - 5$$
This is read as "$f$ of $x$ equals $3x - 5$". It tells us that the function's name is $f$, and its input variable is $x$. 

You will often see different letters used for different functions in the same question to avoid confusion, such as:
* $g(x) = \frac{3x - 4}{5}$
* $h(x) = 2x^2 + 3$

### Domain and Range
Every function has a set of allowed inputs and a set of possible outputs.
* **Domain:** The complete set of all possible **input values** (the $x$-values) that the function is allowed to process.
* **Range:** The complete set of all possible **output values** (the $f(x)$ or $y$-values) that the machine can produce.

## 2. Inverse Functions $f^{-1}(x)$

An inverse function, written as $f^{-1}(x)$, is a machine that does the exact opposite of $f(x)$. It takes the output and brings you back to the original input.

<Aside type="caution" title="Not a Reciprocal!">
A very common misconception is thinking that $f^{-1}(x)$ means $\frac{1}{f(x)}$. **It does not.** The $-1$ here is a notation label meaning "inverse", not a negative index!
</Aside>

### How to Find an Inverse Function Algebraically
There is a reliable, four-step algebraic method to find the inverse of any linear or simple rational function.

**Example:** Find the inverse of $f(x) = \frac{3x - 4}{5}$

1. **Replace $f(x)$ with $y$:** $y = \frac{3x - 4}{5}$
2. **Swap the $x$ and $y$ variables:** $x = \frac{3y - 4}{5}$
3. **Rearrange to make $y$ the subject:**
   $5x = 3y - 4$
   $5x + 4 = 3y$
   $y = \frac{5x + 4}{3}$
4. **Replace $y$ with $f^{-1}(x)$:** $f^{-1}(x) = \frac{5x + 4}{3}$

### Self-Inverse Functions
A function is called **self-inverse** if it is its own inverse. This means that if you perform the four steps above, you end up with the exact same algebraic fraction you started with! In notation, this means:
$$f(x) = f^{-1}(x)$$
Common examples include $f(x) = \frac{1}{x}$ and $f(x) = c - x$ (where $c$ is a constant).

<SteveTip title="The 1-Mark Identity Trick">
Exams love to test if you truly understand what an inverse is without doing any algebra. Because $f$ and $f^{-1}$ cancel each other out perfectly, you must memorise this rule:
$$f(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = x$$
If a question asks you to evaluate $g^{-1}g(42)$ for a horribly complex function, do not calculate it! The answer is simply **42**.
</SteveTip>

## 3. Composite Functions $gf(x)$

A composite function is formed when you substitute one function entirely into another. 

The notation $gf(x)$ means exactly the same thing as $g(f(x))$. You must work from the **inside out**.
1. First, process $x$ through the $f$ machine.
2. Take that result, and process it through the $g$ machine.

You will encounter various combinations. The logic remains the same:
* **$fg(x)$**: Substitute the entirety of $g(x)$ into $f(x)$.
* **$gf(x)$**: Substitute the entirety of $f(x)$ into $g(x)$.
* **$ff(x)$**: Substitute $f(x)$ into itself!
* **$gg(x)$**: Substitute $g(x)$ into itself!

<Aside type="caution" title="Order Matters!">
In algebra, $a \times b$ is the same as $b \times a$. However, with functions, **$fg(x)$ is almost never equal to $gf(x)$**. Pay very close attention to the order of the letters!
</Aside>

## 4. Exponential and Logarithmic Functions (Extended)

In the IGCSE 0607 Extended syllabus, you must be comfortable mixing functions with exponentials and logarithms. Remember the fundamental relationship: **Logarithms and exponentials are inverses of each other.**

* If $f(x) = a^x$, then $f^{-1}(x) = \log_a(x)$
* Base $e$ functions: If $g(x) = e^x$, then $g^{-1}(x) = \ln(x)$
* Base 10 functions: If $h(x) = 10^x$, then $h^{-1}(x) = \log(x)$

This means composite properties apply perfectly here. For example, $f(f^{-1}(x)) = x$ translates to $e^{\ln(x)} = x$, and $\ln(e^x) = x$.

## 5. Solving Equations with Functions

Exam questions frequently ask you to solve equations where composite functions are set equal to a number, or equal to another function (e.g., $fg(x) = h(x)$).

To solve these:
1. First, **form the composite function algebraically** using the substitution steps.
2. Expand and simplify the expression if necessary.
3. Set the simplified expression equal to the other side of the equation.
4. Solve for $x$ using standard algebraic techniques (this often results in a quadratic equation).

<Tabs>
  <TabItem label="📝 Question 1: Self-Inverse Functions">
    Let $f(x) = \frac{x+2}{x-1}$. 
    Find $f^{-1}(x)$ and deduce a special property of this function.
  </TabItem>
  <TabItem label="✅ Solution 1">
    1. Let $y = \frac{x+2}{x-1}$
    2. Swap variables: $x = \frac{y+2}{y-1}$
    3. Rearrange to make $y$ the subject:
       Multiply both sides by $(y-1)$:
       $x(y-1) = y + 2$
       Expand the bracket:
       $xy - x = y + 2$
       Group all the $y$ terms on one side:
       $xy - y = x + 2$
       Factorise out the $y$:
       $y(x - 1) = x + 2$
       Divide by $(x-1)$:
       $y = \frac{x+2}{x-1}$
    4. $f^{-1}(x) = \frac{x+2}{x-1}$
    
    **Deduction:** Because $f^{-1}(x) = f(x)$, this function is **self-inverse**.
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 2: Evaluating Composites (Fractions & Negatives)">
    Let $f(x) = 2x^2 - 3$ and $g(x) = \frac{1}{x}$.
    Find the exact value of $fg(-\frac{1}{2})$.
  </TabItem>
  <TabItem label="✅ Solution 2">
    Work from the inside out. Find $g(-\frac{1}{2})$ first:
    $g(-\frac{1}{2}) = \frac{1}{-\frac{1}{2}}$
    $g(-\frac{1}{2}) = -2$

    Now take this result and substitute it into $f(x)$:
    $f(-2) = 2(-2)^2 - 3$
    $f(-2) = 2(4) - 3$
    $f(-2) = 8 - 3 = 5$

    **Answer:** $fg(-\frac{1}{2}) = 5$
  </TabItem>
</Tabs>

<Tabs>
  <TabItem label="📝 Question 3: Solving Equations with Composites">
    Given $f(x) = 3x - 1$ and $g(x) = x^2$, solve the equation $fg(x) = f(x)$.
  </TabItem>
  <TabItem label="✅ Solution 3">
    **Step 1:** Find the algebraic expression for $fg(x)$.
    Substitute $g(x)$ into $f(x)$:
    $fg(x) = 3(x^2) - 1 = 3x^2 - 1$

    **Step 2:** Set up the equation.
    $fg(x) = f(x)$
    $3x^2 - 1 = 3x - 1$

    **Step 3:** Solve the resulting quadratic equation.
    Add 1 to both sides:
    $3x^2 = 3x$
    Bring terms to one side:
    $3x^2 - 3x = 0$
    Factorise:
    $3x(x - 1) = 0$
    
    **Answers:** $x = 0$ or $x = 1$
  </TabItem>
</Tabs>

<Tabs>
  <TabItem label="📝 Question 4: Exponentials and Logarithms">
    Let $f(x) = e^{2x}$ and $g(x) = \ln(x + 5)$.
    Find $fg(x)$ and give your answer in its simplest form.
  </TabItem>
  <TabItem label="✅ Solution 4">
    Substitute $g(x)$ into the $x$ position of $f(x)$:
    $fg(x) = e^{2(\ln(x + 5))}$

    Use the power law of logarithms ($n\ln(A) = \ln(A^n)$) to move the 2:
    $fg(x) = e^{\ln((x + 5)^2)}$

    Because $e$ and $\ln$ are inverse functions, $e^{\ln(A)} = A$. Therefore, they cancel each other out:
    $fg(x) = (x + 5)^2$

    **Answer:** $fg(x) = (x + 5)^2$ (or $x^2 + 10x + 25$)
  </TabItem>
</Tabs>
