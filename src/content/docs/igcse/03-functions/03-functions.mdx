In mathematics, a function is like a machine: you put a number in, the machine performs a specific set of operations on it, and it spits a new number out. This chapter will teach you how to write functions, reverse them, and chain them together.

---

## 1. Functions, Domain and Range

We use **function notation** to describe these mathematical machines. Instead of writing $y = 3x - 5$, we write:
$$f(x) = 3x - 5$$
This is read as "$f$ of $x$ equals $3x - 5$". It tells us that the function's name is $f$, and its input variable is $x$. 

You will often see different letters used for different functions in the same question to avoid confusion, such as:
* $g(x) = \dfrac{3x - 4}{5}$
* $h(x) = 2x^2 + 3$

### Domain and Range (Mapping Diagram)
Every function has a set of allowed inputs and a set of possible outputs.
* **Domain:** The complete set of all possible **input values** (the $x$-values) that the function is allowed to process.
* **Range:** The complete set of all possible **output values** (the $f(x)$ or $y$-values) that the machine can produce.

## 2. Inverse Functions $f^{-1}(x)$

An inverse function, written as $f^{-1}(x)$, is a machine that does the exact opposite of $f(x)$. It takes the output and brings you back to the original input.

<Aside type="caution" title="Not a Reciprocal!">
A very common misconception is thinking that $f^{-1}(x)$ means $\dfrac{1}{f(x)}$. **It does not.** The $-1$ here is a notation label meaning "inverse", not a negative index!
</Aside>

### The Geometry of an Inverse Function
Visually, the graph of an inverse function $f^{-1}(x)$ is a perfect reflection of the original function $f(x)$ across the diagonal line $y = x$. If a point $(a, b)$ is on the graph of $f(x)$, then the point $(b, a)$ will be on the graph of $f^{-1}(x)$.

### How to Find an Inverse Function Algebraically
There is a reliable algebraic method to find the inverse of any linear or simple rational function.

**Example:** Find the inverse of $f(x) = \dfrac{3x - 4}{5}$

<Steps>
  1. Replace $f(x)$ with $y$:
     $$y = \dfrac{3x - 4}{5}$$
  2. Swap the $x$ and $y$ variables:
     $$x = \dfrac{3y - 4}{5}$$
  3. Multiply both sides by $5$:
     $$5x = 3y - 4$$
  4. Add $4$ to both sides:
     $$5x + 4 = 3y$$
  5. Divide by $3$ to make $y$ the subject again:
     $$\dfrac{5x + 4}{3} = y$$
  6. Replace $y$ with $f^{-1}(x)$:
     $$f^{-1}(x) = \dfrac{5x + 4}{3}$$
</Steps>

### Self-Inverse Functions
A function is called **self-inverse** if it is its own inverse. This means that if you perform the four steps above, you end up with the exact same algebraic fraction you started with! In notation, this means:
$$f(x) = f^{-1}(x)$$
Common examples include $f(x) = \dfrac{1}{x}$ and $f(x) = c - x$ (where $c$ is a constant).

<SteveTip title="The 1-Mark Identity Trick">
Exams love to test if you truly understand what an inverse is without doing any algebra. Because $f$ and $f^{-1}$ cancel each other out perfectly, you must memorise this rule:
$$f(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = x$$
If a question asks you to evaluate $g^{-1}g(42)$ for a horribly complex function, do not calculate it! The answer is simply **42**.
</SteveTip>

## 3. Composite Functions $gf(x)$

A composite function is formed when you substitute one function entirely into another. 

The notation $gf(x)$ means exactly the same thing as $g(f(x))$. You must work from the **inside out**.
1. First, process $x$ through the $f$ machine.
2. Take that result, and process it through the $g$ machine.

You will encounter various combinations. The logic remains the same:
* **$fg(x)$**: Substitute the entirety of $g(x)$ into $f(x)$.
* **$gf(x)$**: Substitute the entirety of $f(x)$ into $g(x)$.
* **$ff(x)$**: Substitute $f(x)$ into itself!
* **$gg(x)$**: Substitute $g(x)$ into itself!

<Aside type="caution" title="Order Matters!">
In algebra, $a \times b$ is the same as $b \times a$. However, with functions, **$fg(x)$ is almost never equal to $gf(x)$**. Pay very close attention to the order of the letters!
</Aside>

## 4. Exponential and Logarithmic Functions (Extended)

In the IGCSE 0607 Extended syllabus, you must be comfortable mixing functions with exponentials and logarithms. Remember the fundamental relationship: **Logarithms and exponentials are inverses of each other.**

* If $f(x) = a^x$, then $f^{-1}(x) = \log_a(x)$
* Base $e$ functions: If $g(x) = e^x$, then $g^{-1}(x) = \ln(x)$
* Base 10 functions: If $h(x) = 10^x$, then $h^{-1}(x) = \log(x)$

This means composite properties apply perfectly here. For example, $f(f^{-1}(x)) = x$ translates to $e^{\ln(x)} = x$, and $\ln(e^x) = x$.

## 5. Solving Equations with Functions

Exam questions frequently ask you to solve equations where composite functions are set equal to a number, or equal to another function (e.g., $fg(x) = h(x)$).

To solve these:
1. First, **form the composite function algebraically** using the substitution steps.
2. Expand and simplify the expression if necessary.
3. Set the simplified expression equal to the other side of the equation.
4. Solve for $x$ using standard algebraic techniques (this often results in a quadratic equation).

## 6. Practice Problems

<Tabs>
  <TabItem label="📝 Question 1: Linear & Rational Functions">
    Let $f(x) = 3x - 5$ and $g(x) = \dfrac{x+2}{x-1}$.
    
    a) Evaluate $f(2)$.

    b) Find an expression for $f^{-1}(x)$.

    c) Find an expression for $g^{-1}(x)$ and state why $g(x)$ is a special type of function.

    d) Solve the equation $f(g(x)) = 4$.
  </TabItem>
  <TabItem label="✅ Solution 1">
    **a)**
    Substitute $2$ into the function:
    $$f(2) = 3(2) - 5 = 1$$

    **b)**
    <Steps>
      1. Let $y$ equal the function:
         $$y = 3x - 5$$
      2. Swap the $x$ and $y$ variables:
         $$x = 3y - 5$$
      3. Add $5$ to both sides:
         $$x + 5 = 3y$$
      4. Divide by $3$:
         $$y = \dfrac{x+5}{3}$$
         $$f^{-1}(x) = \dfrac{x+5}{3}$$
    </Steps>

    **c)**
    <Steps>
      1. Let $y$ equal the function and swap the variables:
         $$x = \dfrac{y+2}{y-1}$$
      2. Multiply both sides by $(y-1)$:
         $$x(y-1) = y + 2$$
      3. Expand the brackets:
         $$xy - x = y + 2$$
      4. Bring all terms containing $y$ to one side:
         $$xy - y = x + 2$$
      5. Factor out $y$:
         $$y(x-1) = x+2$$
      6. Divide by $(x-1)$ to isolate $y$:
         $$y = \dfrac{x+2}{x-1}$$
         $$g^{-1}(x) = \dfrac{x+2}{x-1}$$
    </Steps>
    Because $g^{-1}(x) = g(x)$, it is a **self-inverse** function.

    **d)**
    <Steps>
      1. Substitute $g(x)$ into $f(x)$ and set equal to $4$:
         $$3\left(\dfrac{x+2}{x-1}\right) - 5 = 4$$
      2. Add $5$ to both sides:
         $$3\left(\dfrac{x+2}{x-1}\right) = 9$$
      3. Divide by $3$:
         $$\dfrac{x+2}{x-1} = 3$$
      4. Multiply both sides by $(x-1)$:
         $$x+2 = 3(x-1)$$
      5. Expand and solve for $x$:
         $$x+2 = 3x - 3$$
         $$5 = 2x$$
         $$x = 2.5$$
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 2: Evaluating Composites">
    Let $f(x) = 2x^2 - 3$ and $g(x) = \dfrac{1}{x}$.
    
    a) Find the exact value of $g(4)$.

    b) Find the exact value of $fg(-\dfrac{1}{2})$.

    c) Find an expression for $gf(x)$.

    d) Solve the equation $fg(x) = 5$.
  </TabItem>
  <TabItem label="✅ Solution 2">
    **a)**
    Substitute $4$ into $g(x)$:
    $$g(4) = \dfrac{1}{4}$$

    **b)**
    <Steps>
      1. Work inside out. First, evaluate $g(-\dfrac{1}{2})$:
         $$g\left(-\dfrac{1}{2}\right) = \dfrac{1}{-\dfrac{1}{2}} = -2$$
      2. Substitute this result into $f(x)$:
         $$f(-2) = 2(-2)^2 - 3$$
      3. Calculate the final value:
         $$f(-2) = 2(4) - 3 = 5$$
    </Steps>

    **c)**
    Substitute $f(x)$ completely into the $x$ position of $g(x)$:
    $$gf(x) = \dfrac{1}{2x^2 - 3}$$

    **d)**
    <Steps>
      1. Find $fg(x)$ by substituting $g(x)$ into $f(x)$:
         $$fg(x) = 2\left(\dfrac{1}{x}\right)^2 - 3$$
         $$fg(x) = \dfrac{2}{x^2} - 3$$
      2. Set the composite equal to $5$:
         $$\dfrac{2}{x^2} - 3 = 5$$
      3. Add $3$ to both sides:
         $$\dfrac{2}{x^2} = 8$$
      4. Multiply by $x^2$:
         $$2 = 8x^2$$
      5. Divide by $8$:
         $$x^2 = \dfrac{1}{4}$$
      6. Take the square root (remember the $\pm$):
         $$x = \dfrac{1}{2} \quad \text{or} \quad x = -\dfrac{1}{2}$$
    </Steps>
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 3: Exponential & Logarithmic (Extended)">
    Let $f(x) = e^{2x}$ and $g(x) = \ln(x + 5)$.
    
    a) Evaluate $g(e^2 - 5)$.

    b) Find an expression for $f^{-1}(x)$.

    c) Find an expression for $fg(x)$ in its simplest form.

  <TabItem label="✅ Solution 3">
    **a)**
    <Steps>
      1. Substitute $(e^2 - 5)$ into $g(x)$:
         $$g(e^2 - 5) = \ln((e^2 - 5) + 5)$$
      2. Simplify the terms inside the log:
         $$g(e^2 - 5) = \ln(e^2)$$
      3. Apply inverse properties:
         $$g(e^2 - 5) = 2$$
    </Steps>

    **b)**
    <Steps>
      1. Let $y = e^{2x}$ and swap the variables:
         $$x = e^{2y}$$
      2. Take the natural logarithm of both sides to remove the exponential:
         $$\ln(x) = 2y$$
      3. Divide by $2$:
         $$y = \dfrac{\ln(x)}{2}$$
         $$f^{-1}(x) = \dfrac{\ln(x)}{2}$$
    </Steps>

    **c)**
    <Steps>
      1. Substitute $g(x)$ into $f(x)$:
         $$fg(x) = e^{2\ln(x+5)}$$
      2. Use the power rule for logarithms to move the coefficient inside:
         $$fg(x) = e^{\ln((x+5)^2)}$$
      3. Simplify using the inverse relationship $e^{\ln(A)} = A$:
         $$fg(x) = (x+5)^2$$
    </Steps>

    **d)**
    <Steps>
      1. Set $f(x) = 10$:
         $$e^{2x} = 10$$
      2. Take the natural log of both sides:
         $$2x = \ln(10)$$
      3. Divide by $2$:
         $$x = \dfrac{\ln(10)}{2}$$
      4. Calculate the value to 3 significant figures:
         $$x \approx 1.15$$
    </Steps>
  </TabItem>
</Tabs>

    Let $h(x) = \dfrac{x^3 - 4x^2 + \sin(x)}{\sqrt{x+2}}$ and $k(x) = x + 3$.
    
    a) State the domain of the function $y = \sqrt{x+2}$.

  <TabItem label="✅ Solution 4">
    **a)**
    You cannot evaluate the square root of a negative number in real mathematics.
    <Steps>
      1. Set the expression inside the square root to be greater than or equal to zero:
         $$x+2 \ge 0$$
      2. Solve the inequality:
         $$x \ge -2$$
    </Steps>

    **b)**
    Applying a function and its inverse back-to-back perfectly cancels out the operation. By the identity rule $h^{-1}(h(x)) = x$.
    You do not need to calculate anything. The exact value is $14.8$.

    **c)**
    <Steps>
      1. Substitute $k(x)$ into itself:
         $$kk(x) = k(x+3)$$
      2. Replace $x$ with $(x+3)$ in the expression:
         $$kk(x) = (x + 3) + 3$$
      3. Simplify:
         $$kk(x) = x + 6$$
    </Steps>

    **d)**
    <Steps>
      1. Find $k^{-1}(x)$ by letting $y = x+3$, swapping to $x = y+3$, and rearranging:
         $$k^{-1}(x) = x-3$$
      2. Set the function equal to its inverse:
         $$x+3 = x-3$$
      3. Subtract $x$ from both sides:
         $$3 = -3$$
    </Steps>
    Because $3 = -3$ is a mathematically false statement, there are **no solutions**.

      2. Calculate the value:
         $$f(4) = 1^2 + 2 = 3$$
    </Steps>

    **b)**

         $$f^{-1}(x) = \sqrt{x-2} + 3$$
    </Steps>

    A fundamental rule is that the domain of an inverse function is identical to the range of the original function.
    Since the range of $f(x)$ is $y \ge 2$, the domain of $f^{-1}(x)$ is $x \ge 2$.

  <TabItem label="📝 Question 6: Fractional Equations & Composites">
    Let $f(x) = \dfrac{3x-1}{2}$ and $g(x) = \dfrac{4}{x}$.
    
    a) Find $fg(x)$ in simplest form.

    b) Solve the equation $fg(x) = x$.

    c) Find an expression for $g^{-1}(x)$.

    d) Solve the equation $g(x) = f^{-1}(x)$.
  </TabItem>
  <TabItem label="✅ Solution 6">
    **a)**
    <Steps>
      1. Substitute $g(x)$ into $f(x)$:
         $$fg(x) = \dfrac{3\left(\dfrac{4}{x}\right) - 1}{2}$$
      2. Simplify the numerator:
         $$fg(x) = \dfrac{\dfrac{12}{x} - 1}{2}$$
      3. Multiply the numerator and denominator by $x$ to clear the nested fraction:
         $$fg(x) = \dfrac{12 - x}{2x}$$
    </Steps>

    **b)**
    <Steps>
      1. Set $fg(x)$ equal to $x$:
         $$\dfrac{12 - x}{2x} = x$$
      2. Multiply both sides by $2x$:
         $$12 - x = 2x^2$$
      3. Bring all terms to one side to form a quadratic:
         $$2x^2 + x - 12 = 0$$
      4. Factorise (or use the quadratic formula):
         $$(2x - 3)(x + 4) = 0$$
      5. Solve for $x$:
         $$x = 1.5 \quad \text{or} \quad x = -4$$
    </Steps>

    **c)**
    <Steps>
      1. Let $y = \dfrac{4}{x}$ and swap variables:
         $$x = \dfrac{4}{y}$$
      2. Multiply by $y$ and divide by $x$:
         $$y = \dfrac{4}{x}$$
         $$g^{-1}(x) = \dfrac{4}{x}$$
    </Steps>
    *(Notice it is a self-inverse function).*

    **d)**
    <Steps>
      1. First find $f^{-1}(x)$. Let $y = \dfrac{3x-1}{2}$, swap variables, and isolate $y$:
         $$x = \dfrac{3y-1}{2}$$
         $$2x = 3y-1$$
         $$f^{-1}(x) = \dfrac{2x+1}{3}$$
      2. Set $g(x)$ equal to $f^{-1}(x)$:
         $$\dfrac{4}{x} = \dfrac{2x+1}{3}$$
      3. Cross-multiply:
         $$12 = x(2x+1)$$
      4. Expand and set to zero:
         $$12 = 2x^2 + x$$
         $$2x^2 + x - 12 = 0$$
    </Steps>
    This results in the exact same quadratic equation as part (b). 
    The solutions are $x = 1.5$ or $x = -4$.

      3. Divide by $8$:

  <TabItem label="📝 Question 8: Mixed Function Operations">
    Let $f(x) = 10^x$, $g(x) = \log(x)$, and $h(x) = 2x$.
    
    a) Find $fgh(x)$ in simplest form.

    b) Solve the equation $h(x) = f(2)$.

    c) Find $h^{-1}(x)$.

    d) Show algebraically that $h(x) + h(x) = h(2x)$.
  </TabItem>
  <TabItem label="✅ Solution 8">
    **a)**
    <Steps>
      1. Work from the deepest level out. First, find $gh(x)$:
         $$gh(x) = \log(2x)$$
      2. Substitute this result into $f(x)$:
         $$f(gh(x)) = 10^{\log(2x)}$$
      3. Use inverse properties to simplify. Because $10^x$ and $\log_{10}(x)$ cancel each other out:
         $$fgh(x) = 2x$$
    </Steps>

    **b)**
    <Steps>
      1. Evaluate $f(2)$:
         $$f(2) = 10^2 = 100$$
      2. Set $h(x)$ equal to this value:
         $$2x = 100$$
      3. Divide by $2$:
         $$x = 50$$
    </Steps>

    **c)**
    <Steps>
      1. Let $y = 2x$ and swap variables:
         $$x = 2y$$
      2. Isolate $y$:
         $$y = \dfrac{x}{2}$$
         $$h^{-1}(x) = \dfrac{x}{2}$$
    </Steps>

    **d)**
    <Steps>
      1. Evaluate the Left-Hand Side (LHS) by adding the function to itself:
         $$h(x) + h(x) = 2x + 2x = 4x$$
      2. Evaluate the Right-Hand Side (RHS) by substituting $2x$ into the function:
         $$h(2x) = 2(2x) = 4x$$
      3. Conclude the proof:
         Since LHS = RHS ($4x = 4x$), the algebraic statement is proven true.
    </Steps>
