# 3.1 Graphs of Functions

Understanding the shapes and features of different mathematical functions is a core skill in IGCSE 0607. You must be able to visually identify a function type, extract unknown values from its graph, and understand features like symmetry, amplitude, and period.

---

## 1. Recognising Function Shapes

According to the syllabus, you must be able to recognise the following families of functions purely by looking at their curves.

### (a) Linear: $f(x) = ax + b$

Linear graphs are straight lines. The value $a$ represents the gradient (slope), and $b$ represents the y-intercept.

## 2. Determining Unknown Coefficients

Exam questions frequently provide a graph and its general equation with missing letters (e.g., $f(x) = ax^2 + bx + c$), asking you to find those values.

> **Key Strategies for Extracting Values from Graphs:**
> 1. **The y-intercept:** Look at where the graph crosses the y-axis ($x = 0$). For a quadratic $f(x) = ax^2 + bx + c$, the y-intercept is exactly $c$.
> 2. **The x-intercepts (Roots):** Look at where the graph crosses the x-axis ($y = 0$). Substitute these coordinates into the equation to create simultaneous equations.
> 3. **Amplitude and Period:** For trig graphs like $y = a \sin(bx)$, the peak value gives you $a$. Observe how many degrees it takes to complete one wave to determine $b$.

    Determine the unknown constants and equations for these linear functions:

          const x = i;

          const mathY = 3 * mathX + 4;

      </svg>

    a) A straight line has the equation $f(x) = ax + b$. It crosses the y-axis at $4$ and passes through the point $(3, 13)$. Find the values of $a$ and $b$.

    b) A line passes through the coordinates $(2, 7)$ and $(5, 13)$. Find the equation of the line in the form $f(x) = ax + b$.

    c) The equation of a line is $f(x) = -2x + 8$. Find the exact coordinates of its x-intercept.

    d) A linear graph is parallel to the line $y = 3x - 1$ and passes through the point $(0, 5)$. Write down the equation of this graph.
  </TabItem>
  <TabItem label="✅ Solution 1">
    **a)**
    <Steps>
      1. Find $b$: The y-intercept is $(0, 4)$, so substituting $x = 0$ gives $f(0) = b \implies b = 4$.
      2. Find $a$: Substitute $(3, 13)$ into $f(x) = ax + 4 \implies 13 = 3a + 4 \implies 9 = 3a \implies a = 3$.
    </Steps>

    **b)**
    <Steps>
      1. Find the gradient $a$: $a = \frac{y_2 - y_1}{x_2 - x_1} = \frac{13 - 7}{5 - 2} = \frac{6}{3} = 2$.
      2. Find the y-intercept $b$: Substitute $(2, 7)$ and $a=2$ into $y = ax + b \implies 7 = 2(2) + b \implies 7 = 4 + b \implies b = 3$.
      3. Combine: The equation is $f(x) = 2x + 3$.
    </Steps>

    **c)**
    <Steps>
      1. The x-intercept occurs where $y = 0$.
      2. Set $f(x) = 0 \implies 0 = -2x + 8 \implies 2x = 8 \implies x = 4$.
      3. The coordinates are $(4, 0)$.
    </Steps>

    **d)**
    <Steps>
      1. Parallel lines have the same gradient, so $a = 3$.
      2. The line passes through $(0, 5)$, which is the y-intercept, so $b = 5$.
      3. The equation is $f(x) = 3x + 5$.
    </Steps>
  </TabItem>
</Tabs>

    **b)**
    <Steps>
      1. Substitute $(3, 10)$ into the equation: $10 = a(3)^2 - 8$.
      2. Simplify and solve for $a$: $10 = 9a - 8 \implies 18 = 9a \implies a = 2$.
    </Steps>

    **c)**
    <Steps>
      1. The x-axis intersections occur where $f(x) = 0$.
      2. Set $x^2 - 6x + 5 = 0$.
      3. Factorise the quadratic: $(x - 5)(x - 1) = 0$.
      4. The solutions are $x = 5$ and $x = 1$.
    </Steps>

    **d)**
    <Steps>
      1. Roots are the x-intercepts, where $y = 0$.
      2. Substitute $x = 4$ and $f(x) = 0$ into the equation: $0 = -(4)^2 + c \implies 0 = -16 + c \implies c = 16$.
      3. (Verification: substituting $x = -4$ gives the same result: $0 = -(-4)^2 + 16 \implies 0 = -16 + 16$).
    </Steps>
