<Aside type="note" title="Graph Symmetry">
- **Quadratic functions** have a vertical line of symmetry passing through their vertex (turning point).
- **Cubic functions** often have rotational symmetry about their point of inflection.
- **Reciprocal functions** have rotational symmetry about the origin (if no translations have occurred) and do not cross the axes due to asymptotes.
</Aside>

For any basic exponential function $f(x) = a^x$, the y-intercept is always $(0, 1)$ because any number to the power of $0$ equals $1$. The curve gets extremely close to the x-axis but never touches it (an asymptote at $y = 0$).
</Aside>

## 2. Trigonometric Graphs (0° to 360°)

You must recognise the sine, cosine, and tangent waves. For functions of the form $f(x) = a \sin(bx)$ or $f(x) = a \cos(bx)$:
* **Amplitude ($a$):** The maximum height of the wave measured from the centre line.
* **Period:** The distance on the x-axis for one complete cycle. It is calculated as $\frac{360^\circ}{b}$.

## 3. Determining Unknown Coefficients

Exam questions will frequently provide a graph and its general equation with missing letters (e.g., $f(x) = ax^2 + bx + c$), asking you to find those values.

<Aside type="note" title="Extracting Values from Graphs">
1. **The y-intercept:** Look at where the graph crosses the y-axis ($x = 0$). For a quadratic $f(x) = ax^2 + bx + c$, the y-intercept is exactly $c$. For a linear graph $y = ax + b$, the y-intercept is $b$.
2. **The x-intercepts (Roots):** Look at where the graph crosses the x-axis ($y = 0$). Substitute these coordinates into the equation to create simultaneous equations for $a$ and $b$.
3. **Amplitude and Period:** For trig graphs like $y = a \sin(bx)$, the peak value gives you $a$. Observe how many degrees it takes to complete one wave to determine $b$.
</Aside>

<Aside type="caution" title="Common Misconception: Roots vs. Turning Points">
A common mistake is using the coordinates of the turning point (vertex) when the question expects you to use the roots (where $y=0$). Always verify which key feature the coordinate represents before substituting it into your equation.
</Aside>

---

## 4. Reading Values from a Graph

You can use a given curve to solve equations without any algebra.
* **Finding $f(x)$:** If asked to find the value of a function for a specific $x$, go to that number on the x-axis, draw a vertical line up to the curve, and read across to the y-axis.
* **Solving $f(x) = k$:** If asked to solve an equation, draw a horizontal line at $y = k$. The x-coordinates of every intersection point are your solutions.

## 5. Practice Problems

<Tabs>
  <TabItem label="📝 Question 1: Finding Coefficients (Quadratic)">
    The graph of $f(x) = ax^2 + bx + 5$ passes through the points $(1, 8)$ and $(2, 15)$. 
    Determine the values of $a$ and $b$.
  </TabItem>
  <TabItem label="✅ Solution 1">
    Substitute the given coordinates into the equation to form simultaneous equations.
    
    Using $(1, 8)$:
    $a(1)^2 + b(1) + 5 = 8$
    $a + b = 3$  **(Equation 1)**

    Using $(2, 15)$:
    $a(2)^2 + b(2) + 5 = 15$
    $4a + 2b = 10$
    Divide by 2:
    $2a + b = 5$ **(Equation 2)**

    Subtract Equation 1 from Equation 2:
    $(2a + b) - (a + b) = 5 - 3$
    $a = 2$

    Substitute $a = 2$ back into Equation 1:
    $2 + b = 3 \implies b = 1$

    **Answer:** $a = 2, b = 1$
  </TabItem>
</Tabs>

  <TabItem label="📝 Question 2: Trigonometric Parameters">
    A trigonometric graph has the equation $f(x) = a \cos(bx)$. The graph has a maximum value of 4 at $x=0^\circ$, and its first minimum occurs at $x=90^\circ$. 
    Find the values of $a$ and $b$.
  </TabItem>
  <TabItem label="✅ Solution 2">
    1. **Find $a$:** The maximum value of the cosine wave gives the amplitude directly. Therefore, $a = 4$.
    2. **Find $b$:** The distance from the maximum ($0^\circ$) to the first minimum ($90^\circ$) represents half of a full period.
       If half the period is $90^\circ$, the full period is $180^\circ$.
       We know that Period = $\frac{360^\circ}{b}$.
       $180^\circ = \frac{360^\circ}{b}$
       $b = 2$

    **Answer:** $a = 4, b = 2$
  </TabItem>
</Tabs>

<Tabs>
  <TabItem label="📝 Question 3: Exponential Growth">
    The graph of $f(x) = a^x$ passes through the point $(3, 125)$. 
    Find the value of $a$ and determine whether it represents growth or decay.
  </TabItem>
  <TabItem label="✅ Solution 3">
    Substitute the coordinate $(3, 125)$ into the equation:
    $125 = a^3$
    
    Take the cube root of both sides:
    $a = \sqrt[3]{125} = 5$

    Since $a = 5$ and $5 > 1$, this graph represents **exponential growth**.

    **Answer:** $a = 5$, Growth.
  </TabItem>
