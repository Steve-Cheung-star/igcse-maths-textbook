3.6 Transforming Graphs of Functions

In mathematics, you can take a standard graph and “move” it around the coordinate grid. Sliding a graph up, down, left, or right without changing its fundamental shape or orientation is called a translation.

In the IGCSE 0607 syllabus, you need to understand how adding or subtracting an integer kkk to a function changes its graph, and how to describe this movement using a translation vector.

1. Vertical Translations: y=f(x)+ky = f(x) + ky=f(x)+k

Section titled “1. Vertical Translations: y=f(x)+ky = f(x) + ky=f(x)+k”

When you add or subtract a number on the outside of a function, it affects the yyy-values directly. This causes the entire graph to slide vertically.

  • If you have y=f(x)+ky = f(x) + ky=f(x)+k (where kkk is positive), the graph shifts UP by kkk units.
  • If you have y=f(x)−ky = f(x) - ky=f(x)−k (where kkk is positive), the graph shifts DOWN by kkk units.

Notice how the vertex of the parabola moves exactly according to the number added or subtracted at the end of the equation.

2. Horizontal Translations: y=f(x+k)y = f(x + k)y=f(x+k)

Section titled “2. Horizontal Translations: y=f(x+k)y = f(x + k)y=f(x+k)”

When you add or subtract a number on the inside of the function’s brackets, it affects the xxx-values before the function evaluates them. This causes the entire graph to slide horizontally.

Steve's Exam Wisdom

The Opposite Direction Trap

Horizontal translations are counter-intuitive. They do the opposite of what the sign suggests!

  • A plus sign inside the bracket shifts the graph to the LEFT (the negative direction).
  • A minus sign inside the bracket shifts the graph to the RIGHT (the positive direction).
  • y=f(x+k)y = f(x + k)y=f(x+k) shifts the graph LEFT by kkk units.
  • y=f(x−k)y = f(x - k)y=f(x−k) shifts the graph RIGHT by kkk units.

Let’s look at the base graph of f(x)=x2f(x) = x^2f(x)=x2 again, this time with horizontal translations:

Notice that f(x−4)f(x - 4)f(x−4) shifted the vertex to the positive side (x=4x = 4x=4), while f(x+3)f(x + 3)f(x+3) shifted the vertex to the negative side (x=−3x = -3x=−3).

3. Describing Transformations using Column Vectors

Section titled “3. Describing Transformations using Column Vectors”

In exams, you are frequently asked to describe a translation using a column vector. A column vector looks like this: (pq)\begin{pmatrix} p \\ q \end{pmatrix}(pq​).

  • The top number (ppp) tells you the horizontal movement (left/right).
  • The bottom number (qqq) tells you the vertical movement (up/down).

If a function f(x)f(x)f(x) is translated by the vector (pq)\begin{pmatrix} p \\ q \end{pmatrix}(pq​) to become a new function g(x)g(x)g(x), the formula is: g(x)=f(x−p)+qg(x) = f(x - p) + qg(x)=f(x−p)+q

Connecting the Formula to the Vector

Because of the “opposite direction trap” for horizontal translations, the horizontal shift (ppp) takes a minus sign inside the brackets, but the vertical shift (qqq) stays precisely as it is on the outside.

For example, translating by (−52)\begin{pmatrix} -5 \\ 2 \end{pmatrix}(−52​) means going left 5 and up 2. The new function would be y=f(x+5)+2y = f(x + 5) + 2y=f(x+5)+2.

The graph of y=x2y = x^2y=x2 is mapped onto the graph of y=g(x)y = g(x)y=g(x) by a translation with the vector (3−1)\begin{pmatrix} 3 \\ -1 \end{pmatrix}(3−1​). Find g(x)g(x)g(x) in terms of xxx.

The vector (3−1)\begin{pmatrix} 3 \\ -1 \end{pmatrix}(3−1​) means:

  • Shift right by 3 units (horizontal).
  • Shift down by 1 unit (vertical).

Start with f(x)=x2f(x) = x^2f(x)=x2.

  • Horizontal shift right by 3 means replacing xxx with (x−3)(x - 3)(x−3). This gives (x−3)2(x - 3)^2(x−3)2.
  • Vertical shift down by 1 means subtracting 1 at the end. This gives (x−3)2−1(x - 3)^2 - 1(x−3)2−1.

Answer: g(x)=(x−3)2−1g(x) = (x - 3)^2 - 1g(x)=(x−3)2−1
