3.1 Graphs of Functions

Understanding the shapes and features of different mathematical functions is a core skill in IGCSE 0607. You must be able to visually identify a function type, extract unknown values from its graph, and understand features like symmetry, amplitude, and period.

1. Recognising Function Shapes

Section titled “1. Recognising Function Shapes”

You must be able to recognise the following families of functions purely by looking at their curves.

Linear: f(x)=ax+bf(x) = ax + bf(x)=ax+b

Quadratic: f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c

Cubic: f(x)=ax3+bx2+cx+df(x) = ax^3 + bx^2 + cx + df(x)=ax3+bx2+cx+d

Reciprocal: f(x)=axf(x) = \frac{a}{x}f(x)=xa​

Graph Symmetry

  • Quadratic functions have a vertical line of symmetry passing through their vertex (turning point).
  • Cubic functions often have rotational symmetry about their point of inflection.
  • Reciprocal functions have rotational symmetry about the origin (if no translations have occurred) and do not cross the axes due to asymptotes.

Exponential Graphs

Section titled “Exponential Graphs”

Exponential graphs take the form f(x)=axf(x) = a^xf(x)=ax. Their shape changes drastically depending on whether the base aaa is greater than 1, or between 0 and 1.

Exponential Growth: a>1a > 1a>1

Exponential Decay: 0<a<10 < a < 10<a<1

Exponential Intercepts

For any basic exponential function f(x)=axf(x) = a^xf(x)=ax, the y-intercept is always (0,1)(0, 1)(0,1) because any number to the power of 0 equals 1. The curve gets extremely close to the x-axis but never touches it (an asymptote at y=0y = 0y=0).

2. Trigonometric Graphs (0° to 360°)

Section titled “2. Trigonometric Graphs (0° to 360°)”

You must recognise the sine, cosine, and tangent waves. For functions of the form f(x)=asin⁡(bx)f(x) = a \sin(bx)f(x)=asin(bx) or f(x)=acos⁡(bx)f(x) = a \cos(bx)f(x)=acos(bx):

  • Amplitude (aaa): The maximum height of the wave measured from the centre line.
  • Period: The distance on the x-axis for one complete cycle. It is calculated as 360∘b\frac{360^\circ}{b}b360∘​.

Sine: Starts at 0, goes up.

Cosine: Starts at the maximum.

Tangent: Asymptotes at 90∘,270∘90^\circ, 270^\circ90∘,270∘.

Steve's Exam Wisdom

Tangent Asymptotes

The f(x)=tan⁡(x)f(x) = \tan(x)f(x)=tan(x) graph looks completely different. It does not have an amplitude and it repeats every 180∘180^\circ180∘. Be prepared to draw its vertical asymptotes at 90∘,270∘90^\circ, 270^\circ90∘,270∘, etc.

3. Determining Unknown Coefficients

Section titled “3. Determining Unknown Coefficients”

Exam questions will frequently provide a graph and its general equation with missing letters (e.g., f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c), asking you to find those values.

Extracting Values from Graphs

  1. The y-intercept: Look at where the graph crosses the y-axis (x=0x = 0x=0). For a quadratic f(x)=ax2+bx+cf(x) = ax^2 + bx + cf(x)=ax2+bx+c, the y-intercept is exactly ccc. For a linear graph y=ax+by = ax + by=ax+b, the y-intercept is bbb.
  2. The x-intercepts (Roots): Look at where the graph crosses the x-axis (y=0y = 0y=0). Substitute these coordinates into the equation to create simultaneous equations for aaa and bbb.
  3. Amplitude and Period: For trig graphs like y=asin⁡(bx)y = a \sin(bx)y=asin(bx), the peak value gives you aaa. Observe how many degrees it takes to complete one wave to determine bbb.

Confusing Roots with Turning Points

A common mistake is using the coordinates of the turning point (vertex) when the question expects you to use the roots (where y=0y=0y=0). Always verify which key feature the coordinate represents before substituting it into your equation.

4. Reading Values from a Graph

Section titled “4. Reading Values from a Graph”

You can use a given curve to solve equations without any algebra.

  • Finding f(x)f(x)f(x): If asked to find the value of a function for a specific xxx, go to that number on the x-axis, draw a vertical line up to the curve, and read across to the y-axis.
  • Solving f(x)=kf(x) = kf(x)=k: If asked to solve an equation, draw a horizontal line at y=ky = ky=k. The x-coordinates of every intersection point are your solutions.
