3.3 Functions

In mathematics, a function is like a machine: you put a number in, the machine performs a specific set of operations on it, and it spits a new number out. This chapter will teach you how to write functions, reverse them, and chain them together.

1. Functions, Domain and Range

Section titled “1. Functions, Domain and Range”

We use function notation to describe these mathematical machines. Instead of writing y=3x−5y = 3x - 5y=3x−5, we write: f(x)=3x−5f(x) = 3x - 5f(x)=3x−5 This is read as ”fff of xxx equals 3x−53x - 53x−5”. It tells us that the function’s name is fff, and its input variable is xxx.

You will often see different letters used for different functions in the same question to avoid confusion, such as:

  • g(x)=3x−45g(x) = \frac{3x - 4}{5}g(x)=53x−4​
  • h(x)=2x2+3h(x) = 2x^2 + 3h(x)=2x2+3

Domain and Range

Section titled “Domain and Range”

Every function has a set of allowed inputs and a set of possible outputs.

  • Domain: The complete set of all possible input values (the xxx-values) that the function is allowed to process.
  • Range: The complete set of all possible output values (the f(x)f(x)f(x) or yyy-values) that the machine can produce.

Mapping Diagrams

The visual representation above is called a mapping diagram. It clearly shows which input (Domain) maps to which output (Range). Exam questions may occasionally present functions in this format rather than algebraically.

2. Inverse Functions f−1(x)f^{-1}(x)f−1(x)

Section titled “2. Inverse Functions f−1(x)f^{-1}(x)f−1(x)”

An inverse function, written as f−1(x)f^{-1}(x)f−1(x), is a machine that does the exact opposite of f(x)f(x)f(x). It takes the output and brings you back to the original input.

Not a Reciprocal!

A very common misconception is thinking that f−1(x)f^{-1}(x)f−1(x) means 1f(x)\frac{1}{f(x)}f(x)1​. It does not. The −1-1−1 here is a notation label meaning “inverse”, not a negative index!

How to Find an Inverse Function Algebraically

Section titled “How to Find an Inverse Function Algebraically”

There is a reliable, four-step algebraic method to find the inverse of any linear or simple rational function.

Example: Find the inverse of f(x)=3x−45f(x) = \frac{3x - 4}{5}f(x)=53x−4​

  1. Replace f(x)f(x)f(x) with yyy: y=3x−45y = \frac{3x - 4}{5}y=53x−4​
  2. Swap the xxx and yyy variables: x=3y−45x = \frac{3y - 4}{5}x=53y−4​
  3. Rearrange to make yyy the subject: 5x=3y−45x = 3y - 45x=3y−4 5x+4=3y5x + 4 = 3y5x+4=3y y=5x+43y = \frac{5x + 4}{3}y=35x+4​
  4. Replace yyy with f−1(x)f^{-1}(x)f−1(x): f−1(x)=5x+43f^{-1}(x) = \frac{5x + 4}{3}f−1(x)=35x+4​

Self-Inverse Functions

Section titled “Self-Inverse Functions”

A function is called self-inverse if it is its own inverse. This means that if you perform the four steps above, you end up with the exact same algebraic fraction you started with! In notation, this means: f(x)=f−1(x)f(x) = f^{-1}(x)f(x)=f−1(x) Common examples include f(x)=1xf(x) = \frac{1}{x}f(x)=x1​ and f(x)=c−xf(x) = c - xf(x)=c−x (where ccc is a constant).

Steve's Exam Wisdom

The 1-Mark Identity Trick

Exams love to test if you truly understand what an inverse is without doing any algebra. Because fff and f−1f^{-1}f−1 cancel each other out perfectly, you must memorise this rule: f(f−1(x))=xandf−1(f(x))=xf(f^{-1}(x)) = x \quad \text{and} \quad f^{-1}(f(x)) = xf(f−1(x))=xandf−1(f(x))=x If a question asks you to evaluate g−1g(42)g^{-1}g(42)g−1g(42) for a horribly complex function, do not calculate it! The answer is simply 42.

3. Composite Functions gf(x)gf(x)gf(x)

Section titled “3. Composite Functions gf(x)gf(x)gf(x)”

A composite function is formed when you substitute one function entirely into another.

The notation gf(x)gf(x)gf(x) means exactly the same thing as g(f(x))g(f(x))g(f(x)). You must work from the inside out.

  1. First, process xxx through the fff machine.
  2. Take that result, and process it through the ggg machine.

You will encounter various combinations. The logic remains the same:

  • fg(x)fg(x)fg(x): Substitute the entirety of g(x)g(x)g(x) into f(x)f(x)f(x).
  • gf(x)gf(x)gf(x): Substitute the entirety of f(x)f(x)f(x) into g(x)g(x)g(x).
  • ff(x)ff(x)ff(x): Substitute f(x)f(x)f(x) into itself!
  • gg(x)gg(x)gg(x): Substitute g(x)g(x)g(x) into itself!

Order Matters!

In algebra, a×ba \times ba×b is the same as b×ab \times ab×a. However, with functions, fg(x)fg(x)fg(x) is almost never equal to gf(x)gf(x)gf(x). Pay very close attention to the order of the letters!

4. Exponential and Logarithmic Functions (Extended)

Section titled “4. Exponential and Logarithmic Functions (Extended)”

In the IGCSE 0607 Extended syllabus, you must be comfortable mixing functions with exponentials and logarithms. Remember the fundamental relationship: Logarithms and exponentials are inverses of each other.

  • If f(x)=axf(x) = a^xf(x)=ax, then f−1(x)=log⁡a(x)f^{-1}(x) = \log_a(x)f−1(x)=loga​(x)
  • Base eee functions: If g(x)=exg(x) = e^xg(x)=ex, then g−1(x)=ln⁡(x)g^{-1}(x) = \ln(x)g−1(x)=ln(x)
  • Base 10 functions: If h(x)=10xh(x) = 10^xh(x)=10x, then h−1(x)=log⁡(x)h^{-1}(x) = \log(x)h−1(x)=log(x)

This means composite properties apply perfectly here. For example, f(f−1(x))=xf(f^{-1}(x)) = xf(f−1(x))=x translates to eln⁡(x)=xe^{\ln(x)} = xeln(x)=x, and ln⁡(ex)=x\ln(e^x) = xln(ex)=x.

5. Solving Equations with Functions

Section titled “5. Solving Equations with Functions”

Exam questions frequently ask you to solve equations where composite functions are set equal to a number, or equal to another function (e.g., fg(x)=h(x)fg(x) = h(x)fg(x)=h(x)).

To solve these:

  1. First, form the composite function algebraically using the substitution steps.
  2. Expand and simplify the expression if necessary.
  3. Set the simplified expression equal to the other side of the equation.
  4. Solve for xxx using standard algebraic techniques (this often results in a quadratic equation).
