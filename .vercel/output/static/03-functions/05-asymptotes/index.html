3.5 Asymptotes

When drawing certain mathematical graphs, you will notice that the curve seems to flatten out and approach an invisible boundary line, getting closer and closer to it but never quite touching it. This invisible boundary line is called an asymptote.

In the IGCSE 0607 Extended syllabus, you are not expected to use complex algebra to derive these. Instead, you must understand the concept, identify them visually from a graph or your Graphic Display Calculator (GDC), and know the basic rules for simple functions.

1. What is an Asymptote?

Section titled “1. What is an Asymptote?”

An asymptote is a straight line that a curve approaches as it heads towards infinity. We always draw asymptotes on a graph using dashed lines.

There are two main types you need to identify:

  1. Vertical Asymptotes: Lines parallel to the y-axis. They have the equation x=kx = kx=k. These usually occur when a function attempts to divide by zero.
  2. Horizontal Asymptotes: Lines parallel to the x-axis. They have the equation y=cy = cy=c. These describe the “end behaviour” of a graph as xxx gets extremely large or extremely small.
Steve's Exam Wisdom

Naming the Lines

A very common exam mistake is confusing the equations for horizontal and vertical lines.

  • A vertical line crosses the x-axis, so its equation is always x=numberx = \text{number}x=number.
  • A horizontal line crosses the y-axis, so its equation is always y=numbery = \text{number}y=number.
Steve's Exam Wisdom

Sketching Asymptotes in Exams

Always draw your dashed asymptotes before you sketch the rest of the curve, and always use a pencil! A very common way to lose marks is drawing your curve so carelessly that it accidentally crosses over the asymptote, or curls back on itself. Approach the line, but do not touch or cross it!

2. Rational Functions

Section titled “2. Rational Functions”

Rational functions (fractions with algebra in them) are the most common graphs to feature both horizontal and vertical asymptotes.

Vertical Asymptotes (The Denominator Rule)

Section titled “Vertical Asymptotes (The Denominator Rule)”

A vertical asymptote occurs where a function is undefined. Since you cannot divide by zero in mathematics, simply set the denominator equal to zero and solve for xxx.

For the function f(x)=1x−2+1f(x) = \frac{1}{x - 2} + 1f(x)=x−21​+1:

  • The denominator is x−2x - 2x−2.
  • Set it to zero: x−2=0  ⟹  x=2x - 2 = 0 \implies x = 2x−2=0⟹x=2.
  • The vertical asymptote is the line x=2x = 2x=2.

Functions with Multiple Vertical Asymptotes: If the denominator is a quadratic expression, the function might have two vertical asymptotes! For example, g(x)=5x2−9g(x) = \frac{5}{x^2 - 9}g(x)=x2−95​. Set x2−9=0x^2 - 9 = 0x2−9=0. This gives x2=9x^2 = 9x2=9, which means x=3x = 3x=3 or x=−3x = -3x=−3. This graph has two vertical breaks.

Horizontal Asymptotes (The Ratio Shortcut)

Section titled “Horizontal Asymptotes (The Ratio Shortcut)”

Horizontal asymptotes look at what happens when xxx becomes incredibly large (e.g., 1,000,0001,000,0001,000,000). If you have a rational function in the form y=ax+bcx+dy = \frac{ax + b}{cx + d}y=cx+dax+b​, as xxx gets huge, the constants +b+b+b and +d+d+d become completely insignificant.

The function behaves like y=axcxy = \frac{ax}{cx}y=cxax​, and the xxx‘s cancel out, leaving just the ratio of the coefficients: y=acy = \frac{a}{c}y=ca​.

The Negative Coefficient Trap

Examiners love to scramble the order of the terms. If asked for the horizontal asymptote of h(x)=1−3xx−2h(x) = \frac{1 - 3x}{x - 2}h(x)=x−21−3x​, rewrite the numerator so the xxx is first: h(x)=−3x+1x−2h(x) = \frac{-3x + 1}{x - 2}h(x)=x−2−3x+1​. The coefficients are −3-3−3 and 111. The horizontal asymptote is y=−31y = \frac{-3}{1}y=1−3​, which is y=−3y = -3y=−3.

3. Exponential Asymptotes

Section titled “3. Exponential Asymptotes”

Graphs of the form f(x)=axf(x) = a^xf(x)=ax (such as y=2xy = 2^xy=2x or y=exy = e^xy=ex) do not have vertical asymptotes, but they do have horizontal asymptotes.

The basic y=axy = a^xy=ax curve approaches the x-axis but never quite touches it, because no power of a positive base will ever truly equal zero. Therefore, a standard exponential function has a horizontal asymptote at y=0y = 0y=0.

If the graph is vertically translated, the asymptote moves with it.

As seen above, the function f(x)=2x−3f(x) = 2^x - 3f(x)=2x−3 has been shifted down by 3 units, so its horizontal asymptote is exactly y=−3y = -3y=−3.

4. Trigonometric Asymptotes

Section titled “4. Trigonometric Asymptotes”

While the sine and cosine graphs are continuous waves, the tangent graph (f(x)=tan⁡xf(x) = \tan xf(x)=tanx) is very different. Because tan⁡x=sin⁡xcos⁡x\tan x = \frac{\sin x}{\cos x}tanx=cosxsinx​, whenever cos⁡x=0\cos x = 0cosx=0, the tangent function is attempting to divide by zero, creating a recurring pattern of vertical asymptotes.

The f(x)=tan⁡xf(x) = \tan xf(x)=tanx graph repeats endlessly. Therefore, its vertical asymptotes exist at x=90∘x = 90^\circx=90∘, x=270∘x = 270^\circx=270∘, x=450∘x = 450^\circx=450∘, and so on.

5. Checking Asymptotes with your GDC

Section titled “5. Checking Asymptotes with your GDC”

If you are unsure whether a value is truly a vertical asymptote, you can verify it instantly on your TI-Nspire:

  1. Graph the function.
  2. Press Menu →\rightarrow→ Trace →\rightarrow→ Graph Trace.
  3. Type the xxx-value you suspect is the asymptote (e.g., type 2 and press Enter).
  4. If the bottom-right coordinate displays undef (undefined), you have successfully confirmed your vertical asymptote! (You can also press Ctrl + T to bring up the table and look for undef errors next to your xxx-values).

y=62y=3\begin{aligned} y &= \frac{6}{2} \\ y &= 3 \end{aligned}yy​=26​=3​
