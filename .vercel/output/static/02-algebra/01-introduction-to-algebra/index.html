2.1 Introduction to Algebra

1. Letters as Generalised Numbers (Variables)

Section titled “1. Letters as Generalised Numbers (Variables)”

In arithmetic, we deal with specific, known numbers (like 555, 121212, or −3-3−3). In algebra, we use letters to represent numbers that are either unknown or can change. These letters are called variables.

Using variables allows us to write general rules, formulae, and expressions that work for any number.

The “Invisible” Maths Rules (Algebraic Notation)

Section titled “The “Invisible” Maths Rules (Algebraic Notation)”

To make algebra cleaner and faster to read, mathematicians drop certain symbols. You must memorise these shorthand rules:

  • Multiplication is invisible: Instead of writing 3×a3 \times a3×a, we write 3a\boldsymbol{3a}3a. The number always goes before the letter.
  • Multiplying letters: Instead of a×ba \times ba×b, we write ab\boldsymbol{ab}ab. Usually, letters are written in alphabetical order.
  • Division is a fraction: The ÷\div÷ symbol is rarely used in algebra. Instead of x÷5x \div 5x÷5, we write x5\boldsymbol{\frac{x}{5}}5x​.
  • The invisible “1”: If you have just one xxx, we write x\boldsymbol{x}x, not 1x1x1x. Similarly, −1y-1y−1y is written as −y\boldsymbol{-y}−y.
  • Repeated multiplication: x×xx \times xx×x is written using indices as x2\boldsymbol{x^2}x2.

2. Writing Algebraic Expressions

Section titled “2. Writing Algebraic Expressions”

An expression is a mathematical phrase made up of numbers, variables, and operation symbols (+++, −-−, ×\times×, ÷\div÷). It does not contain an equals sign (===).

You will often need to translate word problems into algebraic expressions. Look for keywords:

  • Addition (+): sum, more than, increased by, total. (e.g., “5 more than xxx” →x+5\rightarrow x + 5→x+5)
  • Subtraction (-): difference, less than, decreased by. (e.g., “3 less than yyy” →y−3\rightarrow y - 3→y−3)
  • Multiplication (×\times×): product, times, double (2×2\times2×), triple (3×3\times3×). (e.g., “double mmm” →2m\rightarrow 2m→2m)
  • Division (÷\div÷): quotient, half (÷2\div 2÷2), shared equally. (e.g., “half of ppp” →p2\rightarrow \frac{p}{2}→2p​)

Examples:

  • ”xxx multiplied by 4, then subtract 7” →4x−7\rightarrow \boldsymbol{4x - 7}→4x−7
  • “The sum of aaa and bbb, divided by 3” →a+b3\rightarrow \boldsymbol{\frac{a + b}{3}}→3a+b​

3. Substitution

Section titled “3. Substitution”

Substitution is the process of replacing variables (letters) in an expression or formula with specific numbers to calculate a final value.

Think of an algebraic expression as a recipe, and substitution as plugging in the specific ingredients you have today.

The Golden Rules of Substitution:

Section titled “The Golden Rules of Substitution:”
  1. Always use brackets when replacing a letter with a number, especially if the number is negative or a fraction.
  2. Follow BIDMAS / BODMAS (Brackets, Indices, Division/Multiplication, Addition/Subtraction) when calculating the final answer.

Example A: Basic Substitution

Section titled “Example A: Basic Substitution”

Evaluate the expression 5x−2y5x - 2y5x−2y when x=4x = 4x=4 and y=3y = 3y=3.

  1. Substitute the numbers using brackets: 5(4)−2(3)5(4) - 2(3)5(4)−2(3)
  2. Multiply: 20−620 - 620−6
  3. Subtract: 14\boldsymbol{14}14

Example B: Substituting Negative Numbers

Section titled “Example B: Substituting Negative Numbers”

Evaluate a2−4ba^2 - 4ba2−4b when a=−3a = -3a=−3 and b=−2b = -2b=−2.

  1. Substitute using brackets: (−3)2−4(−2)(-3)^2 - 4(-2)(−3)2−4(−2)
  2. Evaluate indices: (−3)2=−3×−3=9(-3)^2 = -3 \times -3 = 9(−3)2=−3×−3=9. The expression becomes: 9−4(−2)9 - 4(-2)9−4(−2)
  3. Multiply: −4×−2=+8-4 \times -2 = +8−4×−2=+8. The expression becomes: 9+89 + 89+8
  4. Add: 17\boldsymbol{17}17
Steve's Exam Wisdom

Exam Trick: The Calculator Bracket Trap

Calculators are stupid if you don’t use brackets for certain numbers!

1. Negative Numbers: If you type -3^2 into a calculator, it will calculate −(3×3)=−9-(3 \times 3) = -9−(3×3)=−9. But mathematically, substituting x=−3x = -3x=−3 into x2x^2x2 means (−3)×(−3)(-3) \times (-3)(−3)×(−3), which is +9+9+9. Always type (-3)^2 into your calculator to get the correct positive answer.

2. Fractions: Fractional bases must also be inside brackets! If you substitute x=23x = \frac{2}{3}x=32​ into x2x^2x2 and type 2/3^2 into your calculator, it only squares the 333, giving you 29\frac{2}{9}92​. Always type (2/3)^2 to correctly square the entire fraction and get 49\frac{4}{9}94​.

(Note: Positive whole numbers and decimals are perfectly fine without brackets. Typing 5^2 or 2.5^2 will work exactly as expected!)
